## 💡 Claude Code / MCP 関連

- **Claude Code — Opus 4.8 + Dynamic Workflows 搭載**:
  Opus 4.8 がデフォルト high-effort モードで動作。`/effort xhigh` で最高努力レベルを指定可能。`ultracode` 設定でさらに高い effort を Claude が自動判断。Dynamic Workflows により数十〜数百のバックグラウンドエージェントを並列オーケストレーション可能に。
  - 情報源: [Releasebot - Claude Code June 2026](https://releasebot.io/updates/anthropic/claude-code)

- **MCP タイムアウト設定バグ修正**: 1000ms 未満のサーバータイムアウト設定値が 1秒のウォッチドッグに切り上げられてすべてのツール呼び出しが失敗していた問題を修正。1000ms 未満の値は `MCP_TOOL_TIMEOUT` またはデフォルト値にフォールバックするよう変更。
  - 情報源: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

- **並列ツール呼び出しの堅牢化**: 同一バッチ内で Bash コマンドが失敗しても、他の並列ツール呼び出しがキャンセルされなくなった。各ツールが独立して結果を返す。
  - 情報源: [Releasebot - Claude Code](https://releasebot.io/updates/anthropic/claude-code)

- **`claude agents --json` に `waitingFor` フィールド追加**: 待機中のセッションが何でブロックされているかを JSON 出力で確認できるように。
  - 情報源: [Releasebot - Claude Code](https://releasebot.io/updates/anthropic/claude-code)

- **`/mcp` UI 改善**: 未サインインの claude.ai コネクタが「Show unused connectors」に折りたたまれ、アクティブなコネクタのみ表示されるようクリーンアップ。
  - 情報源: [Releasebot - Claude Code](https://releasebot.io/updates/anthropic/claude-code)
