## 💡 Claude Code / MCP 関連

- **【セキュリティ注意】MCP通信のハイジャック脆弱性が発覚（6月8日）**
  Mitiga Labsの研究者が、Claude CodeのMCP（Model Context Protocol）トラフィックを攻撃者のインフラ経由にリダイレクトする5ステップの攻撃チェーンを実証。悪意ある npm パッケージを起点に、Jira・Confluence・GitHub 等へのOAuth Bearer Tokenを窃取できる。Anthropicからのパッチはまだ出ていない。
  - 情報源: [Hackers Can Hijack Claude Code MCP Traffic to Steal OAuth Tokens](https://cybersecuritynews.com/claude-code-mcp-traffic-hijack/)

- **MCP Developer Platform にプライベートネットワーク対応「MCP Tunnels」追加（Research Preview）**
  Anthropicの開発者プラットフォームがMCPトンネル機能をリサーチプレビューとして公開。プライベートネットワーク内やセルフホストサンドボックスのMCPサーバーへ接続可能に。
  - 情報源: [Claude Developer Platform Updates by Anthropic - June 2026 - Releasebot](https://releasebot.io/updates/anthropic/claude-developer-platform)

- **MCP エコシステムが急拡大**: 公開サーバー数が500超、MCP SDKの月間ダウンロード数が9,700万回を突破。GitHub・Context7・Playwright・Sentry・PostgreSQL・Linear・Supabaseの7つがコア開発ツールとして注目。
  - 情報源: [The MCP Ecosystem in 2026: How the Model Context Protocol Became the Universal Standard](https://chatforest.com/guides/mcp-ecosystem-2026-state-of-the-standard/)

- **Strava が MCP コネクタを提供開始（6月1日）**: StravaがAnthropicのClaude連携用MCPコネクタを公開。サブスクライバーはStravaアカウントを接続し、個人アクティビティデータを会話形式で分析可能に。
  - 情報源: [Strava Launches MCP Connector for Claude Integration](https://letsdatascience.com/news/strava-launches-mcp-connector-for-claude-integration-909e07a0)
