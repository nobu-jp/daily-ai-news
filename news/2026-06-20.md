### Fable 5・Mythos 5 アクセス停止が継続中
- 6月12日夜（ET 17:21）、米国政府の輸出管理指令により **Fable 5** および **Mythos 5** へのアクセスが無効化。外国籍ユーザー（Anthropic社員含む）への提供停止を命じた。
- 政府側はFable 5のジェイルブレイク手法を把握しており、既知の軽微な脆弱性への悪用デモを確認したとされる。
- 6月17〜18日のG7首脳会議（エヴィアン）でトランプ大統領がダリオ・アモデイCEOと会談。ホワイトハウスが「安全保障上の懸念が解消に向かっている」とシグナルを発しており、近日中の復旧が期待される。
- 出典: [Anthropic 公式声明](https://www.anthropic.com/news/fable-mythos-access) / [InfoQ](https://www.infoq.com/news/2026/06/claude-5-release/) / [Softonic](https://en.softonic.com/articles/anthropic-says-fable-5-and-mythos-5-could-return-within-days-freeze-called-temporary)

## 💡 Claude Code / MCP 関連

### v2.1.185（2026-06-20・本日リリース）
- APIレスポンス待機中のヒントメッセージを改善。無応答判定を10秒→20秒に延長し、リトライ表示をよりわかりやすい文言に変更。
- 出典: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

### v2.1.183（2026-06-19）
- **Auto モードの安全性強化**: 以下の破壊的コマンドを明示的に指示されていない限りブロック
  - `git reset --hard`, `git checkout -- .`, `git clean -fd`, `git stash drop`
  - 自分のセッションが行っていないコミットへの `git commit --amend`
  - `terraform destroy`, `pulumi destroy`, `cdk destroy`
- **モデル廃止警告**: 廃止・自動更新されたモデルを指定した場合に警告を表示（stderr/printモード対応）
- **設定改善**: `attribution.sessionUrl` 設定追加（コミット・PRからセッションURLを省略可能）、`/config --help` 追加
- 出典: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

### v2.1.181（2026-06-17）
- `/config key=value` 構文でプロンプトから設定変更が可能に（例: `/config thinking=false`）
- macOS サンドボックス環境での Apple Events サポート（`sandbox.allowAppleEvents` オプトイン）
- `CLAUDE_CODE_PRESENCE_FILE` 環境変数によりPC操作中のモバイル通知を抑制可能
- Bun ランタイムを1.4にアップグレード、長文段落のストリーミング性能向上
- 出典: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

### Claude Managed Agents がプライベート MCP サーバーに接続可能に
- Claude Managed Agents がユーザー管理のサンドボックス内で動作し、プライベートMCPサーバーへの接続をサポート。
- 出典: [Releasebot - Claude Code Updates](https://releasebot.io/updates/anthropic/claude-code)
