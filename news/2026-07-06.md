- **Enterprise 機能強化**: 管理者向けに詳細なアナリティクス・モデルレベルの権限管理・支出アラートが追加。[Anthropic Developer Platform](https://platform.claude.com/docs/en/release-notes/overview)

## 💡 Claude Code / MCP 関連

- **MCPセキュリティ強化**: リポジトリの `.claude/settings.json` 経由でセルフ承認したMCPサーバーを無効化。信頼されていないワークスペースでは ⏸ 承認待ち 表示。[Claude Code Updates - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **MCP認証CLIコマンド追加**: `claude mcp login <n>` / `claude mcp logout <n>` でインタラクティブメニューを開かずにMCPサーバー認証が可能に。SSH経由の `--no-browser` 標準入力リダイレクトにも対応。[Claude Code Changelog](https://code.claude.com/docs/en/changelog)

- **MCP Tunnels（Research Preview）**: プライベートネットワーク内のMCPサーバーへの接続を可能にするMCPトンネル機能がリサーチプレビューとして提供開始。[Claude Developer Platform](https://platform.claude.com/docs/en/release-notes/overview)

- **スラッシュスキル複数呼び出し対応**: `/skill-a /skill-b do XYZ` のように複数のスキルを重ねて呼び出す記法が、最大5件まで連続ロード可能になった。[Claude Code Changelog](https://github.com/anthropics/claude-code/blob/main/CHANGELOG.md)
