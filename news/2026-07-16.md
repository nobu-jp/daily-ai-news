# 🌅 AI朝刊 - 2026年7月16日

## 🤖 Claude / Anthropic アップデート

- **月次レポート機能（Beta）**: `Settings > Reflect` で、Claudeとの会話の振り返りが可能に。最も多く話したトピック・最もアクティブだった曜日・ピーク時間帯などを表示。Free / Pro / Max プランのWeb版・Claude Desktopで提供（メモリ有効時）。([Releasebot](https://releasebot.io/updates/anthropic/claude))
- **Microsoft 365 書き込みツール追加**: Microsoft 365 コネクタに書き込みツールが追加。メールの下書き・送信・整理、カレンダー管理、OneDrive / SharePoint へのファイル作成・更新が可能に。([Releasebot](https://releasebot.io/updates/anthropic/claude))
- **HIPAA 対応の自己管理フロー**: Claude Enterprise および Claude Platform（API）向けに、HIPAA 対応の設定を管理者が自己完結で行えるフローが追加。BAA のレビュー、実装ガイドのダウンロード、HIPAA 設定の有効化を一括で実施可能。([Releasebot](https://releasebot.io/updates/anthropic))
- **API キー有効期限設定**: Claude Console でAPIキー作成時に有効期限を設定できるように。期限前のメールリマインダー・管理者向けレポートも追加。([Claude Developer Platform](https://releasebot.io/updates/anthropic/claude-developer-platform))
- **Admin API（Beta）**: Claude Enterprise 組織のメンバー管理が Admin API で可能に。メンバー一覧・招待・ロール変更・グループ管理などをプログラムから操作できる。([Releasebot](https://releasebot.io/updates/anthropic))
- **Claude for Teachers 開始**: 米国 K-12 の認定教員に対し、Claude のプレミアムツールへの無料アクセスを提供するプログラムを開始。全50州の教育基準に対応したカリキュラム連携・AI リテラシートレーニングも含む。([Releasebot](https://releasebot.io/updates/anthropic/claude))
- **Alibaba が Anthropic ツールを社内禁止**: Alibaba が「蒸留攻撃（distillation attack）」を理由に、従業員による Anthropic AI（Claude）の使用を禁止したと報道（7月6日）。([CNBC](https://www.cnbc.com/2026/07/06/alibaba-anthropic-ai-ban-claude-china.html))

## 💡 Claude Code / MCP 関連

- **最新バージョン v2.1.211**（2026年7月15日時点）。([Claude Code Changelog](https://code.claude.com/docs/en/changelog))
- **サブエージェントテキストストリーミング追加**: `--forward-subagent-text` フラグ（または `CLAUDE_CODE_FORWARD_SUBAGENT_TEXT` 環境変数）を使用すると、サブエージェントのテキスト・思考内容を `stream-json` 出力に含められるように。([Releasebot](https://releasebot.io/updates/anthropic/claude-code))
- **MCP サーバー修正**:
  - `--mcp-config` または `.mcp.json` 経由で設定された MCP サーバーが、サーバーごとの `request_timeout_ms` を無視して60秒デフォルトでタイムアウトしていたバグを修正。
  - OAuth MCP サーバーでトークンリフレッシュ失敗後に毎回手動再認証が必要になっていた問題を修正。
  - `url` キーはあるが `type` がない設定に対するエラーメッセージを改善（`"type": "http"` を提案する内容に）。
  ([Releasebot](https://releasebot.io/updates/anthropic/claude-code))
- **"Claude Browser" サーバー名を予約**: `Claude Browser` という MCP サーバー名を予約（`Claude Preview` に続き）。ユーザー設定の MCP サーバーはこの名前を使用不可に。([Claude Code Changelog](https://code.claude.com/docs/en/changelog))
- **その他 UX 改善**: スクリーンリーダーモード追加、Vim インサートモードのキーリマップ対応、マウスサポート、`/doctor` コマンドの強化。([Releasebot](https://releasebot.io/updates/anthropic/claude-code))
