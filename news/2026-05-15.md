## 💡 Claude Code / MCP 関連

- **`workspace` が MCP の予約済みサーバー名に**
  同名のサーバーは警告とともにスキップされる。また、MCP サーバーの再接続時にツール一覧を大量出力しなくなり、サーバープレフィックス単位でサマリー表示へ変更。
  - 情報源: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

- **MCP サーバーの起動エラー時に自動リトライ（最大 3 回）**
  一時的なエラーで切断されたまま放置されるケースを解消。
  - 情報源: [Releasebot - Claude Code](https://releasebot.io/updates/anthropic/claude-code)

- **`/mcp` コマンドがツール数・接続状態を表示**
  接続済みサーバーのツール数を表示し、ツール数 0 のサーバーにはフラグを立てて可視化。
  - 情報源: [Claude Code May 2026 Release Notes](https://pasqualepillitteri.it/en/news/2223/claude-code-may-2026-release-notes-radio-plugin-marketplace)

- **MCP stdio サーバーの引数破損バグ修正**
  `CLAUDE_CODE_SHELL_PREFIX` 設定時にスペースやシェルメタ文字を含む引数が破損する問題を修正。
  - 情報源: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

- **Claude Code 大型アップデート：フック・プラグイン・エージェント制御を強化**
  新しいフック・プラグインオプション、エージェント/セッション制御の改善、権限フロー・バックグラウンド処理・MCP ハンドリングの全面改善。プラグインマーケットプレイスも登場。
  - 情報源: [Claude Code Changelog](https://claudefa.st/blog/guide/changelog)
