- **Claude for Small Business 提供開始**: QuickBooks・PayPal・HubSpot・Canva・Google Workspace・Microsoft 365 等と連携した業務自動化ワークフロー
  - 出典: [Anthropic News](https://www.anthropic.com/news)

- **Claude Platform on AWS**: AWS の課金・認証基盤で Claude API を完全利用可能に
  - 出典: [Anthropic News](https://www.anthropic.com/news)

- **Microsoft 365 アドイン GA**: Excel・PowerPoint・Word 向けアドインが一般提供開始、Outlook は有料プランで Public Beta 公開中
  - 出典: [Anthropic News](https://www.anthropic.com/news)

- **Gates Foundation パートナーシップ**: 4年間で2億ドルの投資コミットメント
  - 出典: [Anthropic News](https://www.anthropic.com/news)

## 💡 Claude Code / MCP 関連

- **Claude Code 週間利用制限を50%引き上げ（〜7/13まで）**: Pro・Max・Team・Enterprise の全プランで上限を50%増量（5/13〜7/13）

- **Fast モードが Opus 4.7 をデフォルトに変更**（従来は Opus 4.6）

- **`/resume` コマンドがバックグラウンドセッションに対応**: `claude --bg` または agent view で起動したセッションをインタラクティブセッション一覧に表示（`bg` マーク付き）
  - 出典: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

- **MCP スタートアップ高速化**: SDK/ヘッドレス MCP の事前待機をオーバーラップ処理に変更し、遅い MCP サーバーでも最大2秒高速化
  - 出典: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

- **MCP ページネーションバグ修正**: `resources/list`・`resources/templates/list`・`prompts/list` で2ページ目以降のアイテムが欠落していた問題を解消
  - 出典: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

- **`/plugin` 画面の強化**: インストール前にプラグインのコマンド・エージェント・スキル・フック・MCP/LSP サーバー情報を確認可能に
  - 出典: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)
