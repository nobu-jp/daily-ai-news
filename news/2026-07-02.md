## 💡 Claude Code / MCP 関連

- **`/rewind` コマンド追加**: `/clear` 前の会話に戻れる `/rewind` コマンドが追加。誤って会話をクリアしてしまった際の復元が可能に。
  - 情報源: [Claude Code Updates - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **MCP OAuth 不具合修正**: GitLab セルフホストや企業 IdP で発生していた `invalid_scope` エラー（スコープ未指定時に全スコープカタログをリクエストしてしまう問題）を修正。`headersHelper` 認証がツール呼び出しで 401/403 が返った際に自動的に再実行・再接続するよう改善。
  - 情報源: [Claude Code Changelog - claudefa.st](https://claudefa.st/blog/guide/changelog)

- **ストリーミング中のCPU使用率を約37%削減**: Claude Code のストリーミング中の CPU 負荷が大幅に改善。
  - 情報源: [Claude Code Updates - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **エンタープライズ向け MCP コネクター一元管理（Okta 対応）**: 管理者が Okta などの IdP を通じて MCP コネクターを組織全体にプロビジョニングし、ユーザーが初回ログインでゼロタッチアクセスを得られるベータ機能を開始。Claude Chat・Claude Code・Cowork 全体で認証を一元化。Team・Enterprise プランが対象。
  - 情報源: [Claude Platform Docs - Release Notes](https://platform.claude.com/docs/en/release-notes/overview)
