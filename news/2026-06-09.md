## 💡 Claude Code / MCP 関連

- **Claude Code セーフモード追加**: 新たに `/cd` コマンドおよびセーフモードを追加。バンドルスキルの非表示設定も可能に。リモートセッション・バックグラウンドエージェント・エンタープライズポリシー適用も改善。CPU使用量の削減とトラブルシューティングの容易化も実施。
  - 出典: [Releasebot - Claude Code Updates](https://releasebot.io/updates/anthropic/claude-code)

- **MCPタイムアウト設定のバグ修正**: 1000ms未満のper-serverタイムアウト設定が強制的に1秒に切り上げられていた問題を修正。1000ms未満の値は無視してMCP_TOOL_TIMEOUTまたはデフォルトにフォールバックするよう変更。
  - 出典: [Claude Code changelog](https://code.claude.com/docs/en/changelog)

- **エンタープライズMCPポリシー適用の修正**: `allowedMcpServers`/`deniedMcpServers` が再接続時・IDEタイプ設定・初回インストール直後に適用されなかった問題を修正。
  - 出典: [Releasebot - Anthropic Updates](https://releasebot.io/updates/anthropic)

- **Managed AgentsがプライベートMCPサーバーへ接続可能に**: Claude Managed Agentsが企業管理のサンドボックス内でプライベートMCPサーバーに接続できるようになり、エンタープライズ環境でのエージェント活用が拡大。
  - 出典: [Claude Agent SDK - Totalum Blog](https://www.totalum.app/blog/claude-agent-sdk-totalum-2026)

- **法律領域向けMCPコネクタ20種以上追加**: 法律事務所・企業内法務向けに、リサーチ・契約・ディスカバリー・案件管理・法律支援の各ワークフローに対応した20種以上のMCPコネクタと12のプラクティスエリアプラグインを公開。
  - 出典: [Releasebot - Claude Updates](https://releasebot.io/updates/anthropic/claude)
