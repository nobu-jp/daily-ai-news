# 🌅 AI朝刊 - 2026-03-24

## 🤖 Claude / Anthropic アップデート

- **Claude Code「Auto Mode」リサーチプレビュー公開**（3/24）
  AIが人間の承認を待たずに安全と判断したアクションを自律実行する「Auto Mode」が研究プレビューとして公開。現在は Claude Sonnet 4.6 / Opus 4.6 のみ対応、「隔離された環境」での使用を推奨。
  → [TechCrunch](https://techcrunch.com/2026/03/24/anthropic-hands-claude-code-more-control-but-keeps-it-on-a-leash/)

- **Claude Code「Bare Mode」＆チャンネルベース権限リレー追加**（3/21）
  スクリプト向け `-p` 呼び出し用の bare mode と、チャンネル経由で権限承認をスマートフォンへ転送できる仕組みを追加。OAuth・音声モード・セッション・Windows関連のバグも修正。
  → [Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **Claude Code Channels 発表**（3/20）
  Discord・Telegram と Claude Code を連携し、外出先からコード生成を指示できる非同期エージェント連携機能が登場。
  → [VentureBeat](https://venturebeat.com/orchestration/anthropic-just-shipped-an-openclaw-killer-called-claude-code-channels)

- **Claude API: Web検索ツールが一般提供（GA）へ**
  ベータヘッダー不要でウェブ検索・ウェブフェッチが利用可能に。動的フィルタリング対応でトークンコストも削減。
  → [Claude API Docs](https://platform.claude.com/docs/en/release-notes/overview)

- **Claude 3月プロモーション実施中**（〜3/27）
  オフピーク時間帯の利用上限を全プラン（Free/Pro/Max/Team）で2倍に。Claude Code・Excel・PowerPoint アドインも対象。
  → [Evermx](https://www.evermx.com/case/claude-march-2026-usage-promotion-double-limits)

- **Claude Haiku 3 廃止予定**（4/19）
  `claude-3-haiku-20240307` が4月19日に廃止。Claude Haiku 4.5 への移行を推奨。
  → [Claude API Docs](https://platform.claude.com/docs/en/release-notes/overview)
