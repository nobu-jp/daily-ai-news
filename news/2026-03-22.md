## 💡 Claude Code / MCP 関連

- **MCP Elicitation（タスク中の構造化入力要求）**
  MCPサーバーがタスク実行中にインタラクティブなダイアログでユーザーへの入力を要求できる「Elicitation」機能が追加。`Elicitation`・`ElicitationResult`フックで応答の傍受・上書きが可能。
  [Claude Code MCP Docs](https://code.claude.com/docs/en/mcp)

- **MCPツールのオンデマンドロード（MCP Tool Search）**
  MCPサーバーが多数設定されている場合のコンテキストウィンドウ圧迫を解決する機能。ツール定義を事前ロードせず、Claudeが必要なときに動的ロードする仕組み。
  [Releasebot - Claude Code](https://releasebot.io/updates/anthropic/claude-code)

- **Google ColabがオープンソースMCPサーバーを公式公開**（3月19日）
  Claude Code等からColabノートブックをリモートランタイムとして操作可能に。GPUを持つColab環境をローカルのAIエージェントから利用できる。
  [MarkTechPost](https://www.marktechpost.com/2026/03/19/google-colab-now-has-an-open-source-mcp-model-context-protocol-server-use-colab-runtimes-with-gpus-from-any-local-ai-agent/)

- **Kali LinuxがClaude AI（MCP経由）でペネトレーションテストに対応**
  自然言語コマンドをClaude AIが解釈し、MCP経由でKali Linux環境のターミナルコマンドとして実行できるネイティブワークフローを導入。
  [Cybersecurity News](https://cybersecuritynews.com/kali-linux-integrates-claude-ai/)

- **MCPがLinux Foundation傘下「Agentic AI Foundation（AAIF）」に採択**
  ベンダー中立のガバナンスのもとでMCP関連OSSを共同管理する枠組みが整備。MCPエコシステムの標準化が加速。
  [arpable.com](https://arpable.com/technology/anthropic-mcp-ai-data-integration/)

- **MCPエコシステム規模：SDK月間ダウンロード9,700万回超・公開サーバー10,000以上**
  Jira、Confluence、Zapier、Cloudflare、Sentry、PayPal、Linearなどが公式リモートMCPサーバーとして対応済み。
