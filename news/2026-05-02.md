## 💡 Claude Code / MCP 関連

- **Claude Code 大型アップデート**: スマートなモデル選択、プロジェクト状態削除コマンド（`claude project purge`）、強化されたパーミッション制御、OAuth ログイン改善、Windows/PowerShell 修正、テレメトリ改善など多数。
  - `/model` ピッカーが `ANTHROPIC_BASE_URL` を Anthropic 互換ゲートウェイに向けている場合、そのゲートウェイの `/v1/models` エンドポイントからモデル一覧を取得するように改良。
  - 情報源: [Claude Code Updates - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **MCP の改善**:
  - `alwaysLoad` オプション追加: MCP サーバー設定で `alwaysLoad: true` にすると、そのサーバーのツール全件がツール検索の遅延なく常時利用可能に。
  - 起動時に一時エラーが発生した MCP サーバーが最大3回自動リトライするように変更（従来は切断状態のまま）。
  - セッション開始後に nonblocking モードで接続した MCP ツールが ToolSearch で見つからないバグを修正。
  - 情報源: [Claude Code Updates - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **Hooks 強化**: `PostToolUse` フックが全ツールの出力置換（`hookSpecificOutput.updatedToolOutput`）をサポート（従来は MCP ツールのみ）。
  - 情報源: [Claude Code Updates - Releasebot](https://releasebot.io/updates/anthropic/claude-code)
