## 💡 Claude Code / MCP 関連

- **MCP ツール結果の永続化（v2.1.91）**: `_meta["anthropic/maxResultSizeChars"]` アノテーションにより最大 500K 文字まで MCP ツール結果をトランケートせず保持可能に。DB スキーマなど大きなレスポンスが通過できるよう改善。
  - 情報源: [Claude Code v2.1.91 Release Notes (ClaudeWorld)](https://claude-world.com/articles/claude-code-2191-release/)

- **`/powerup` インタラクティブレッスン追加**: Claude Code に学習用の `/powerup` コマンドを追加。再開（resume）パフォーマンスやダイアログ・フック・編集・スクロール・PowerShell 権限などの多数の修正も含む。
  - 情報源: [Claude Code by Anthropic Release Notes (Releasebot)](https://releasebot.io/updates/anthropic/claude-code)

- **`disableSkillShellExecution` 設定の追加**: スキル・カスタムスラッシュコマンド・プラグインコマンド内のインラインシェル実行を無効化できる新設定。
  - 情報源: [Changelog - Claude Code Docs](https://code.claude.com/docs/en/changelog)

- **MCP OAuth が RFC 9728 準拠に**: Protected Resource Metadata discovery に対応し、認可サーバーの検出が標準化。
  - 情報源: [Changelog - Claude Code Docs](https://code.claude.com/docs/en/changelog)

- **MCP パフォーマンス改善**: ターンごとの MCP ツールスキーマの `JSON.stringify` を排除、SSE トランスポートの大規模フレーム処理を O(n²) → O(n) に改善、長い会話での SDK セッション速度低下を修正。
  - 情報源: [Claude Code Changelog (ClaudeLog)](https://claudelog.com/claude-code-changelog/)
