## 💡 Claude Code / MCP 関連

- **ネスト型サブエージェント対応**: サブエージェントが自身のサブエージェントを最大5階層まで生成可能に（バージョン 2.1.170）。
  - 情報源: [Claude Code Updates - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

  - 情報源: [Claude Code changelog](https://code.claude.com/docs/en/changelog)

- **Amazon Bedrock のリージョン自動検出**: `AWS_REGION` 未設定時に `~/.aws` 設定ファイルから自動検出するよう改善。`/status` でリージョン取得元を確認可能。
  - 情報源: [Claude Code Updates - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **プラグイン検索バー追加**: `/plugin` でマーケットプレイスのプラグインを参照する際に検索バーが追加され、探しやすくなった。
