## 💡 Claude Code / MCP 関連

- **MCPエリシテーション（Elicitation）サポート（v2.1.76〜, 2026年3月14日）**: MCPサーバーがタスク実行中にフォームやブラウザURLを表示して構造化入力を要求できるようになった。`ElicitationResult` フックで応答のインターセプト・上書きも可能。
  - 情報源: [Claude Code March 2026 Updates - pasqualepillitteri.it](https://pasqualepillitteri.it/en/news/381/claude-code-march-2026-updates)

- **MCPツール説明を2KBに上限設定**: OpenAPI生成サーバーによるコンテキスト肥大化を防ぐため、MCPのツール説明とサーバー指示を2KBにキャップ。claude.aiコネクタとローカル設定の重複MCPはローカル設定が優先される。
  - 情報源: [Claude Code by Anthropic - Release Notes - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **`--bare` フラグと `--channels` パーミッションリレー**: スクリプト向け `-p` 呼び出し用の `--bare` フラグ（フック・LSP・プラグイン同期をスキップ）と、チャンネルサーバー経由でツール承認プロンプトをスマホに転送する `--channels` パーミッションリレーが追加。
  - 情報源: [Changelog - Claude Code Docs](https://code.claude.com/docs/en/changelog)

- **起動速度改善**: 未認証HTTP/SSE MCPサーバー使用時の `claude -p` 起動が約600ms高速化。REPLもMCPサーバーの接続完了を待たずに即時レンダリングされるように。
  - 情報源: [Claude Code Changelog - claudefa.st](https://claudefa.st/blog/guide/changelog)
