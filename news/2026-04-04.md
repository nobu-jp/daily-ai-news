## 💡 Claude Code / MCP 関連

- **MCPツール検索（Tool Search）機能**を搭載。MCPツールの説明文がコンテキストウィンドウの10%を超えた場合に自動起動し、全ツールを事前読み込みする代わりにオンデマンドで動的ロード。コンテキスト消費を**最大85%削減**（51,000トークン → 8,500トークン）。
  - [Claude Code Just Cut MCP Context Bloat by 46.9% | Medium](https://medium.com/@joe.njenga/claude-code-just-cut-mcp-context-bloat-by-46-9-51k-tokens-down-to-8-5k-with-new-tool-search-ddf9e905f734)

- MCPツールの説明文とサーバー指示を **2KB上限** でキャップ。OpenAPI生成サーバーによるコンテキスト肥大化を防止。

- ローカル設定と claude.ai コネクター（Slack、Gmailなど）で重複設定されたMCPサーバーを自動で重複排除（ローカル設定が優先）。

- MCPツール出力が **10,000トークン超過** 時に警告表示。上限は最大25,000トークン（設定可能）。

- その他：Windowsサポート改善、高速起動・キャッシュ、`--print` モードのバグ修正。
  - [Claude Code Release Notes](https://releasebot.io/updates/anthropic/claude-code)
