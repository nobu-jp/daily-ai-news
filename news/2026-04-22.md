# 🌅 AI朝刊 - 2026年4月22日

## 🤖 Claude / Anthropic アップデート

- **Claude Opus 4.7 が正式リリース（GA）**
  高解像度ビジョン（最大2,576px）、Effort Controls（standard / high / maximum）、Task Budgets（サブタスクへのトークン上限設定）などを搭載。ソフトウェアエンジニアリングや長時間の複雑なコーディングタスクが向上。
  ([GitHub Changelog](https://github.blog/changelog/2026-04-16-claude-opus-4-7-is-generally-available/))

- **Claude Design をリリース（4/17 リサーチプレビュー）**
  Anthropic Labs の新ツール。デザイン・プロトタイプ・スライド・ワンページャーをClaudeで生成可能。Claude Pro / Max / Team / Enterprise 向けに段階的ロールアウト中（追加料金なし）。
  ([Nadcab Labs 解説](https://www.nadcab.com/blog/anthropic-launches-claude-design))

- **Claude Managed Agents パブリックベータ公開**
  セキュアなサンドボックス・組み込みツール・SSEストリーミングを備えた完全マネージドなエージェントハーネス。
  ([Anthropic リリースノート](https://platform.claude.com/docs/en/release-notes/overview))

- **ant CLI リリース**
  Claude API を高速に操作するためのコマンドラインクライアント。Claude Code とのネイティブ統合、APIリソースの YAML バージョン管理をサポート。
  ([Anthropic リリースノート](https://platform.claude.com/docs/en/release-notes/overview))
