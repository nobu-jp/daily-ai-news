- **Cowork & Claude Code にコンピューター使用機能を追加**: Pro・Maxプランのユーザーがファイルを開く・開発ツールの実行・画面上のナビゲーションなどをClaudeに委任可能に。

- **Claude Cowork が macOS / Windows で一般提供開始**: 拡張アナリティクス・OpenTelemetryサポート・エンタープライズ向けロールベースアクセス制御（RBAC）を搭載。
  - 情報源: [Releasebot - Anthropic Release Notes](https://releasebot.io/updates/anthropic)

- **Claude Mythos Preview 発表**: Project Glasswing の一環として、防衛的サイバーセキュリティ研究向けに招待制のゲートリサーチプレビューとして公開。
