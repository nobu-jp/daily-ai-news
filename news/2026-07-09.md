## 💡 Claude Code / MCP 関連

- **MCP `add-from-claude-desktop` のバグ修正** — サーバー名に非対応文字が含まれる場合にコマンドがフリーズする問題を修正。無効な名前はレポートされ、残りのサーバーはインポートが継続される。
  - 情報源: [Claude Code Updates - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **MCPエラーメッセージの改善** — `url` が設定されているが `type` が未指定の場合に、より分かりやすいエラーメッセージ（`"type": "http"` を提案）を表示するよう改善。

- **MCP roots/list にセッションの作業ディレクトリを追加** — セッションの追加作業ディレクトリが MCP の `roots/list` に含まれるようになり、変更時は `notifications/roots/list_changed` が送信される。

- **認証が必要なMCPサーバーのスタートアップ通知** — 起動時に認証が必要なMCPサーバーがある場合、`/mcp` へ誘導する通知を表示。

- **Claude Code エージェントワークフロー強化** — auto モードの安全性向上、Windows の安定性改善、トランスクリプト保護、PR リンクとバックグラウンドタスク処理の改善、`/doctor` コマンドによるセットアップ診断機能を追加。
  - 情報源: [Claude Code Changelog (July 2026)](https://www.gradually.ai/en/changelogs/claude-code/)
