# 🌅 AI朝刊 - 2026-07-03

## 🤖 Claude / Anthropic アップデート

- **Claude Sonnet 5 がデフォルトモデルに**: 7月1日より、Free・Pro 全ユーザーのデフォルトモデルとして Claude Sonnet 5 が正式採用。Opus 4.8 に迫る性能を持ちながら、8月31日まで入門価格 $2/$10 per MTok で提供中。コンテキストウィンドウ 1M トークン・最大出力 128k トークンに対応。
  - 情報源: [Claude Updates - Releasebot](https://releasebot.io/updates/anthropic/claude)

- **Claude Enterprise 管理機能強化**: 豊富な管理者向けアナリティクス、モデルレベルのエンタイトルメント、支出アラート機能が追加。管理者がモデルアクセスを細かく制御しながら、利用状況・コスト・生産性トレンドを可視化できる体制が整備された。
  - 情報源: [Anthropic Release Notes - Releasebot](https://releasebot.io/updates/anthropic)

- **Claude Platform レート制限を大幅引き上げ**: APIスペンドに基づく複雑なティア体系を廃止し、シンプルな新ティア体系へ移行。最上位ティアでは最新 Sonnet・Haiku モデルのレート制限が旧来の5倍に拡大。
  - 情報源: [AI News Today July 3 2026 - BuildFastWithAI](https://www.buildfastwithai.com/blogs/ai-news-today-july-3-2026)

- **Claude apps gateway for Amazon Bedrock・Google Cloud**: セルフホスト型 Claude Code コントロールプレーンが登場。企業 SSO・ポリシー一元管理・ロールベースアクセス・ユーザー別コスト追跡・支出上限設定に対応。
  - 情報源: [Anthropic Release Notes - Releasebot](https://releasebot.io/updates/anthropic)

## 💡 Claude Code / MCP 関連

- **Claude Code にフルスクリーンマウスクリック操作を追加**: ターミナル内でのマウス操作が改善され、フルスクリーン時のクリック制御が可能に。音声入力（ボイスディクテーション）とプラグインマッチングのバグも修正。
  - 情報源: [Claude Code Updates - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **Claude in Chrome が一般提供（GA）開始**: ブラウザ内エージェントワークフローを強化。バックグラウンド通知・ドラフトPRハンドオフ・フェイルオーバー改善などを含む。
  - 情報源: [Claude Code Updates - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **Artifacts が Pro・Max プランで利用可能に**: Claude Code がプライベートなインタラクティブページ（PR ウォークスルーやダッシュボードなど）をビルド・公開・更新しながら作業を継続できる Artifacts 機能が、Pro・Max プランに拡大。
  - 情報源: [AI News Today July 3 2026 - BuildFastWithAI](https://www.buildfastwithai.com/blogs/ai-news-today-july-3-2026)

- **エンタープライズ向け MCP コネクター管理（Okta 対応）**: 管理者が Okta などの IdP を通じて MCP コネクターを組織全体にプロビジョニングし、ユーザーが初回ログインでゼロタッチアクセスを取得できるベータ機能が継続強化中。Claude Chat・Claude Code・Cowork 全体で認証を一元化。
  - 情報源: [Claude Platform Docs - Release Notes](https://platform.claude.com/docs/en/release-notes/overview)

- **MCP セキュリティ改善**: `claude mcp list/get` が、リポジトリ自身が `.claude/settings.json` で承認した `.mcp.json` サーバーを自動起動しなくなった。信頼されていないワークスペースでは ⏸ 承認待ち として表示されるよう変更。
  - 情報源: [Claude Code Changelog - claudefa.st](https://claudefa.st/blog/guide/changelog)
