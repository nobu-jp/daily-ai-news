# 🌅 AI朝刊 - 2026-05-03

## 🤖 Claude / Anthropic アップデート

- **Claude Opus 4.7 リリース**: 複雑な推論とエージェント型コーディングに向けた最も高性能なGAモデルとして発表。([Anthropic Release Notes](https://releasebot.io/updates/anthropic))
- **1Mトークンコンテキストベータ終了**: Sonnet 4.5 / Claude 4 向けの1Mトークンコンテキストベータが終了。後継として Claude Sonnet 4.6 および Claude Opus 4.6 でGAとして提供。([Claude Platform Docs](https://platform.claude.com/docs/en/release-notes/overview))
- **クリエイティブツール連携**: Adobe・Blender・Ableton・Affinity・Autodesk Fusion 向けのClaudeコネクターを追加。既存ツールとの統合を強化。([Anthropic Release Notes](https://releasebot.io/updates/anthropic))
- **Claude Security パブリックベータ開始**: Claude Enterprise 向けにセキュリティ機能がパブリックベータとして提供開始。([Anthropic Release Notes](https://releasebot.io/updates/anthropic))
- **Anthropic ARR 300億ドル突破**: 年間換算売上高が300億ドルを超え、年初の90億ドルから急拡大。100万ドル以上支出する顧客数も約2ヶ月で500社→1000社に倍増。([AI News May 3](https://asanify.com/blog/news/ai-back-office-automation-may-3-2026/))

## 📰 AI業界ニュース TOP3

1. **DeepSeek V4 Pro 評価レポート（NIST）**: 米国立標準技術研究所（NIST）の機関が DeepSeek V4 Pro を評価。米国主要クローズドモデルと比べて約8ヶ月の能力差があり、性能面では GPT-5 に近いが、コスト効率は GPT-5.4 mini より7指標中5つで優れるとのこと。([AI News - aiandnews.com](https://www.aiandnews.com/blog/latest-ai-news-may-2026-2/))

2. **中国がAIアンソロポモーフィック規制を最終化**: 中国が感情的仮想アシスタントや「コンパニオンボット」に対する規制を確定。2026年7月15日施行で、依存性モニタリングや感情状態チェックが義務付けられる。([crescendo.ai](https://www.crescendo.ai/news/latest-ai-news-and-updates))

3. **米国防総省、7つの大手テック企業とAI契約締結**: Anthropicを一時排除した後、国防総省が7社の大手テック企業とAI活用の契約を締結。([CNN Business](https://www.cnn.com/2026/05/01/tech/pentagon-ai-anthropic))

## 💡 Claude Code / MCP 関連

- **MCPサーバー自動リトライ**: 起動時の一時的なエラーに対し、MCP サーバーが最大3回自動リトライするように改善。切断状態のまま放置されなくなった。([Anthropic Claude Code Updates](https://releasebot.io/updates/anthropic/claude-code))
- **MCP list_changed 通知サポート**: MCP サーバーが再接続なしに利用可能なツール・プロンプト・リソースを動的に更新できる `list_changed` 通知をサポート。([Claude Code Changelog](https://claudefa.st/blog/guide/changelog))
- **PostToolUse フックの拡張**: `PostToolUse` フックで `hookSpecificOutput.updatedToolOutput` を使って全ツールの出力を差し替え可能に（以前は MCP 専用）。([Anthropic Claude Code Updates](https://releasebot.io/updates/anthropic/claude-code))
- **Meta 公式 MCP リリース**: Meta が Facebook・Instagram 広告キャンペーンを Claude で管理できる公式 MCP を公開。([Claude Code MCP Guide](https://gist.github.com/septimlabs-code/1f52e699a4a6fbe9c29621b670b958d1))
- **/ultrareview パブリックリサーチプレビュー**: バグ検出エージェント群がクラウドで動作し、結果を CLI またはデスクトップに自動返却する `/ultrareview` がプレビュー公開。([Releasebot Claude Code](https://releasebot.io/updates/anthropic/claude-code))
- **Bedrock サービスティア選択対応**: Claude Code が AWS Bedrock のサービスティア選択をサポート。([Releasebot Claude Code](https://releasebot.io/updates/anthropic/claude-code))
