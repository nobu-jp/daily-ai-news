## 💡 Claude Code / MCP 関連

- **Claude Code の新デフォルトモデルが Opus 4.8 に**：Claude Codeでも Opus 4.8 がデフォルト採用。`fast mode` が低価格で高速化。Dynamic Workflows（並列サブエージェントでの計画・実行・検証）も追加。
  - 情報源: [Releasebot - Claude Code June 2026](https://releasebot.io/updates/anthropic/claude-code)

- **MCP バグ修正（v2.1.161〜162、6月2〜3日）**：
  - `MCP per-server timeout` が1,000ms未満の値をウォッチドッグが切り捨てていた問題を修正。1,000ms未満の設定は無視され、`MCP_TOOL_TIMEOUT` またはデフォルト値にフォールバック。
  - LSP ツールの `workspaceSymbol` が結果を返さなかった問題を修正（`query` パラメーターを受け取りLSPサーバーに正しく渡すよう改修）。
  - 情報源: [Claude Code Changelog](https://code.claude.com/docs/en/changelog) / [GitHub Releases](https://github.com/anthropics/claude-code/releases)

- **並列ツール呼び出しの改善**：Bashコマンドが失敗しても、同一バッチ内の他のツール呼び出しがキャンセルされなくなった。各ツールが独立した結果を返すように改善。
  - 情報源: [Releasebot - Anthropic June 2026](https://releasebot.io/updates/anthropic)

- **`/mcp` コネクタービューの改善**：未サインインの claude.ai コネクターが「未使用のコネクターを表示」行に折りたたまれ、UIがすっきり。
  - 情報源: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)
