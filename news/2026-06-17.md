## 💡 Claude Code / MCP 関連

- **セーフモード追加**：`--safe-mode` フラグまたは環境変数 `CLAUDE_CODE_SAFE_MODE` で起動すると、CLAUDE.md・スキル・プラグイン・フック・MCPサーバー・カスタムコマンドをすべて無効化して起動可能。トラブルシューティング時に有用。
  - 情報源: [jangwook.net](https://jangwook.net/en/blog/en/claude-code-june-2026-new-features-changelog-developer-guide/)

- **フォールバックモデル設定**：`fallbackModel` 設定で最大3つのフォールバックモデルを順番に指定可能。プライマリモデルが過負荷・利用不可時に自動切り替え。`--fallback-model` フラグはインタラクティブセッションにも対応。
  - 情報源: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

- **MCPポリシー強化**：deny ルールのツール名位置でグロブパターン（`*`で全ツール拒否）をサポート。クロスセッションメッセージのセキュリティを強化し、他の Claude セッションから中継されたメッセージはユーザー権限を持たないよう変更。
  - 情報源: [Releasebot Claude Code](https://releasebot.io/updates/anthropic/claude-code)

- **ポストセッションフック追加** および `/cd` コマンドで作業ディレクトリの変更が可能に。
  - 情報源: [Releasebot Claude Code](https://releasebot.io/updates/anthropic/claude-code)
