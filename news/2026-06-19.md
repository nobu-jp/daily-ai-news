## 🤖 Claude / Anthropic アップデート

- **Claude Code v2.1.183 リリース（本日）** — 破壊的なgitコマンド（`git reset --hard` など）を自動ブロックする安全機能を追加。`terraform destroy` / `pulumi destroy` も特定スタック指定時のみ実行可能に。`/config key=value` 構文で設定をプロンプトから直接変更できる機能も追加。
  - 情報源: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

- **Fable 5 / Mythos 5 輸出規制対応（継続中）** — 6月12日に米政府の輸出規制指令を受け、Anthropicが両モデルへの海外からのアクセスを停止。引き金はAmazon研究者がFable 5でコードベースの脆弱性発見ツール化できることをホワイトハウスに報告したこと。韓国最大の通信会社でAnthropicの$1億投資家であるSK Telecomeが中国安全保障リスクとして名指しされたことも判明。6月18日のソウルオフィス開設イベントで国際担当MDが「数日以内に復旧見込み」と表明。
  - 情報源: [CNBC](https://www.cnbc.com/2026/06/12/anthropic-disables-access-to-fable-5-and-mythos-5-to-comply-with-government-directive.html) / [Tom's Hardware](https://www.tomshardware.com/tech-industry/artificial-intelligence/sk-telecom-named-as-the-korean-carrier-at-the-center-of-anthropics-mythos-export-controls) / [Korea JoongAng Daily](https://www.koreajoongangdaily.com/business/anthropic-confident-of-reenabling-mythos-fable-5-access-in-coming-days-executive/12727522)

- **Claude Design × Claude Code 統合** — デザインアシスタント「Claude Design」がローカルコードベースから作業を開始し、Claude Codeにハンドオフしてインターフェースを実装できるように。スクラッチからの実装不要。
  - 情報源: [Engadget](https://www.engadget.com/2196329/anthropics-design-assistant-now-works-better-with-its-coding-agent/)

- **エンタープライズMCPコネクター（Okta統合）** — 管理者がOkta経由でMCPコネクターを組織全体にプロビジョニング可能に。ユーザーは初回ログイン時にゼロタッチでアクセス取得。
  - 情報源: [Releasebot - Anthropic](https://releasebot.io/updates/anthropic)

- **Claude Platform on AWS 提供開始** — Claude APIがAnthropicマネージドインフラでAWS経由で利用可能に。AWSビリングおよびIAM認証に対応。
  - 情報源: [Releasebot - Anthropic](https://releasebot.io/updates/anthropic)

## 💡 Claude Code / MCP 関連

- **v2.1.183（2026-06-19）主なアップデート:**
  - 破壊的なgit操作（`git reset --hard`、`git checkout -- .`）を自動ブロック（ユーザーが明示的に削除を指示した場合を除く）
  - `/config key=value` 構文でプロンプトから任意の設定を変更可能（例: `/config thinking=false`）
  - `attribution.sessionUrl` 設定追加でコミット・PRからclaude.aiリンクの除外が可能
  - 思考ブロックのみを返すターンの無限ループバグを修正
  - MCP認証が必要なサーバーがツールを誤公開する問題を修正
  - WebSearchがサブエージェントで空の結果を返す問題を修正
  - 情報源: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

- **v2.1.181（2026-06-17）:**
  - `CLAUDE_CLIENT_PRESENCE_FILE` 環境変数でモバイル通知を抑制可能
  - Bunランタイムを1.4にアップグレード
  - 長い段落のストリーミング表示が行ごとに改善

- **v2.1.178（2026-06-15）Agent Teams改善:**
  - `TeamCreate`/`TeamDelete`ツールを廃止し、Agentツールの`name`パラメータから直接チームメイトをスポーン可能に
  - ネストされた `.claude/skills` ディレクトリからスキルを自動ロード

- **エンタープライズMCP管理（Okta連携）:** 組織全体へのコネクター一括プロビジョニングが可能に
  - 情報源: [Releasebot - Claude Code](https://releasebot.io/updates/anthropic/claude-code)
