# 🌅 AI朝刊 - 2026年6月2日

## 🤖 Claude / Anthropic アップデート

- **Claude Opus 4.8 リリース**: コーディング・エージェントタスク・プロフェッショナル向け作業の強化が中心。Opus 4.7 より応答が速く、長いセッションでのコンテキスト保持と指示追従が大幅に改善。
  - 情報源: [Introducing Claude Opus 4.8 - Anthropic](https://www.anthropic.com/news/claude-opus-4-8)

- **モデル廃止予告（6/15）**: Claude Sonnet 4 および Claude Opus 4 が 2026年6月15日に Claude API から退役予定。後継モデルへの移行を推奨。
  - 情報源: [Anthropic's 2026 Claude Updates - TJ Digital](https://tjrobertson.com/anthropic-2026-claude-updates/)

- **サブスクリプション変更（6/15〜）**: 6月15日より、Claude サブスクリプションプラン経由のプログラム的利用が別の月次クレジットプールに移行。個人・法人ともに影響あり。
  - 情報源: [Anthropic Splits Claude Subscriptions - devtoolpicks](https://devtoolpicks.com/blog/anthropic-splits-claude-subscriptions-agent-sdk-credit-june-2026)

- **Claude Platform on AWS 提供開始**: Anthropic 管理インフラとして AWS 経由で Claude API が利用可能に。AWS Billing・IAM 認証に対応。
  - 情報源: [Claude Platform Release Notes](https://platform.claude.com/docs/en/release-notes/overview)

- **Claude Managed Agents 強化**: セルフホスト型サンドボックス（パブリックベータ）開始。Cloudflare・Daytona・Modal・Vercel など外部環境でのツール実行が可能に。マルチエージェントセッション・Outcomes もパブリックベータへ。
  - 情報源: [Claude Platform Release Notes](https://platform.claude.com/docs/en/release-notes/overview)

- **`ant` CLI リリース**: Claude API 向けコマンドラインクライアント。Claude Code とのネイティブ統合、YAML による API リソースのバージョン管理が可能。
  - 情報源: [Claude Platform Release Notes](https://platform.claude.com/docs/en/release-notes/overview)

## 💡 Claude Code / MCP 関連

- **Claude Code v2.1.153（5/28リリース）の新機能**:
  - GitHub/Git プラグインの `skipLfs` オプション追加
  - npm グローバルインストール時の自動更新失敗を通知する one-time ノーティス実装

  - 情報源: [Claude Platform Release Notes](https://platform.claude.com/docs/en/release-notes/overview)

- **Strava MCP コネクター一般提供開始（6/1〜）**:
  Strava が公式 MCP を Strava サブスクリプションの一部として提供開始。セキュリティ・品質ガイドライン付きで構築。
  - 情報源: [Strava MCP Connector – Strava Support](https://support.strava.com/hc/en-us/articles/46190267796237-Strava-MCP-Connector)
