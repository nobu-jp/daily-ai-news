## 💡 Claude Code / MCP 関連

- **MCP タイムアウトバグ修正**：`--mcp-config` または `.mcp.json` で設定した `request_timeout_ms` が新規セッションで無視され、60秒デフォルトで失敗していた問題を修正。
  - 情報源: [Claude Code Updates July 2026](https://releasebot.io/updates/anthropic/claude-code)

- **OAuth MCPサーバーの再認証ループ修正**：トークンリフレッシュに一度失敗すると手動再認証が必要になっていたバグを解消。

- **`/doctor` コマンドが修正機能を搭載**（`/checkup` とも呼ばれる）：読み取り専用レポートの表示にとどまらず、問題を診断・自動修正できるように進化。未使用スキル・MCPサーバー・プラグインのコンテキストコスト分析も追加。
  - 情報源: [Claude Code /checkup Explained - MCP.Directory](https://mcp.directory/blog/claude-code-checkup-command-2026)

- **MCP エラーメッセージの改善**：サーバー設定に `url` はあるが `type` がない場合、`"type": "http"` を提案するより明確なエラーを表示するよう改善。

- **MCP サーバー予約名の追加**：`"Claude Browser"` を予約済みサーバー名に追加（`"Claude Preview"` と同様）。ユーザー設定のMCPサーバーはこれらの名前を使用不可に。
  - 情報源: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)
