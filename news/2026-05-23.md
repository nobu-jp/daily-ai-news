## 💡 Claude Code / MCP 関連

- **MCP ページネーション修正**：2ページ目以降で MCP サーバーのリソース・テンプレート・プロンプトが欠落するバグを修正。
  - 情報源: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

- **MCP 起動速度改善**：SDK/ヘッドレス MCP 起動時の pre-wait が最初のターン前をブロックせず起動と並行して実行されるよう変更。遅い MCP サーバーで最大2秒の高速化。
  - 情報源: [Claude Code Updates May 2026 - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **Fast Mode のデフォルトモデルが Opus 4.7 に変更**：Fast mode が Opus 4.7 をデフォルトに。4.6 に固定したい場合は `CLAUDE_CODE_OPUS_4_6_FAST_MODE_OVERRIDE=1` で上書き可能。
  - 情報源: [Claude Code Changelog - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **`claude agents` コマンドに新フラグ追加**：`--add-dir`, `--settings`, `--mcp-config`, `--plugin-dir`, `--permission-mode`, `--model`, `--effort`, `--dangerously-skip-permissions` が追加され、バックグラウンドセッションの細かい設定が可能に。
  - 情報源: [Claude Code Updates May 2026 - Releasebot](https://releasebot.io/updates/anthropic/claude-code)
