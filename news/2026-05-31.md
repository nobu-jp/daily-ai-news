## 💡 Claude Code / MCP 関連

- **Claude Code v2.1.152 リリース（5月24日前後）**：Opus 4.8 対応・高負荷デフォルト設定・Dynamic Workflows・Chrome / MCP / プラグイン連携の強化が含まれる。
  - [Claude Code v2.1.152 Major Updates - DevelopersIO](https://dev.classmethod.jp/en/articles/20260524-claude-code-updates-v2-1-152/)
  - [Claude Code Updates - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **Dynamic Workflows（新機能）**：大規模タスクを数十〜数百のエージェントに並列分散して処理する「ダイナミックワークフロー」が追加。従来では困難だった超大規模問題への対応が可能に。

- **MCP バグ修正**：`/mcp` ツール一覧の長いツール名・複数行説明のレンダリング問題を修正。Stdio MCP サーバーのサブプロセスが `CLAUDE_CODE_SESSION_ID` と `CLAUDECODE=1` を環境変数として受け取れるよう改善。未承認の `.mcp.json` サーバーが自動接続ではなく「Pending approval」として表示されるように変更。
  - [Claude Code Changelog - Anthropic GitHub](https://github.com/anthropics/claude-code/blob/main/CHANGELOG.md)
  - [Claude Code Docs Changelog](https://code.claude.com/docs/en/changelog)

- **法律向け MCP コネクター20件以上追加**：リサーチ・契約・ディスカバリー・案件管理・法律援助など、法律事務所向けの MCP コネクター20件超＋12の実務領域プラグインを新たにリリース。
