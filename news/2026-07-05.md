## 💡 Claude Code / MCP 関連

- **MCP 仕様リリース候補（RC）発表 — 最大改訂版がほぼ確定**: 2026年7月28日付でMCP（Model Context Protocol）の最大規模改訂となるリリース候補が公表。主な変更点:
  - **MCP Apps**: サーバーがサンドボックス化されたiframe内にインタラクティブなHTML UIを配信できる新機能。すべてのUI操作は従来のツール呼び出しと同じ審査・同意フローを通過。
  - **Tasks拡張**: 長時間実行タスクのサポートを追加。
  - **Streamable HTTP改善**: `Mcp-Method` と `Mcp-Name` ヘッダーが必須化され、ロードバランサやゲートウェイがボディを検査せずにルーティング可能に。
  - **キャッシュ制御**: リスト/リソース読み取り結果に `ttlMs` と `cacheScope` を追加（HTTP Cache-Controlを参考にした設計）。
  - **OAuth/OpenID Connect** との整合性強化。
  - 情報源: [MCP Blog - 2026-07-28 Release Candidate](https://blog.modelcontextprotocol.io/posts/2026-07-28-release-candidate/)

- **MCP公式レジストリ、サーバー数9,400超に**: MCPプロトコルはバージョン1.0安定版として普及が進み、公式レジストリのサーバー数が9,400を超えた。OpenAI・Google・主要IDEがプロトコルをサポート。
  - 情報源: [MCP in 2026: complete guide](https://agence-scroll.com/en/blog/mcp-claude-guide-2026)
