## 💡 Claude Code / MCP 関連

- **MCP `add-from-claude-desktop` のバグ修正**: サーバー名に非対応の文字が含まれる場合に処理が止まる問題を修正。無効な名前を報告しつつ残りのサーバーのインポートを続行するよう改善。
  - 情報源: [Claude Code Updates by Anthropic - July 2026](https://releasebot.io/updates/anthropic/claude-code)

- **MCP サーバー名の予約**: 「Claude Browser」を予約名に追加（「Claude Preview」に加えて）。ユーザー設定のMCPサーバーはこれらの名前を使用不可に。
  - 情報源: [Claude Code Updates by Anthropic - July 2026](https://releasebot.io/updates/anthropic/claude-code)

- **MCPエラーメッセージの改善**: `url` のみで `type` が指定されていないサーバー設定に対し、`"type": "http"` を提案するより明確なエラーメッセージを表示。
  - 情報源: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

- **MCP `request_timeout_ms` の修正**: `--mcp-config` や `.mcp.json` 経由で設定したサーバーがデフォルト60秒でタイムアウトしていた問題を修正。長時間ツール呼び出しが正しくタイムアウト設定に従うよう改善。
  - 情報源: [Claude Code Updates by Anthropic - July 2026](https://releasebot.io/updates/anthropic/claude-code)

- **ヘッドレスセッションでのフックストリーミング修正**: SessionStart フックでフックイベントがストリーミングされない問題を修正。リモートワーカーがフック中にアイドル回収される問題を解消。
  - 情報源: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

- **パフォーマンス改善**: ストリーミング応答性の向上・起動時メモリの削減・VS Code リモートコントロール設定の追加。
  - 情報源: [Claude Code Updates by Anthropic - July 2026](https://releasebot.io/updates/anthropic/claude-code)

## 📖 用語解説

| 用語 | 解説 |
|------|------|
| コンテキストウィンドウ | AIモデルが一度に処理できるテキストの最大量。「100万トークン」とは、約75万語に相当する大量のテキストを一度に扱えることを意味する。 |
| MCPトンネル | Model Context Protocol（MCP）を使って、プライベートなネットワーク上にあるサーバーへ安全に接続するための仕組み。社内システムとClaudeを繋げる用途に使われる。 |
| エージェンティックワークロード | AIが自律的に複数のタスクを連続して実行する作業のこと。例えば、コード生成→テスト→デプロイを自動で行うような処理を指す。 |
| Intelligence Index | Artificial Analysisが公表するAIモデルの総合能力ランキング。各モデルの推論・コーディング・知識など複数の指標を総合したスコアで順位づけする。 |
| ChatGPT Work | OpenAIがリリースしたビジネス向けの統合AIプラットフォーム。ChatGPTとCodexを統合し、職場の生産性向上に特化した機能を提供する。 |
| Cowork | Claudeのリモート共同作業機能。複数デバイスからチャット・プロジェクト・アーティファクトを共有し、チームでリアルタイムに作業できる環境を提供する。 |
| request_timeout_ms | MCPサーバーへのリクエストがタイムアウトするまでの時間（ミリ秒単位）をカスタム設定するオプション。デフォルトは60秒。 |
| SessionStart フック | Claude Codeのセッション開始時に自動的に実行されるスクリプトやコマンド。環境セットアップや事前チェックなどに利用される。 |
