## 💡 Claude Code / MCP 関連

  ([releasebot.io](https://releasebot.io/updates/anthropic/claude-code))

- **プラグイン依存関係の管理を強化**
  `claude plugin disable` は依存プラグインが存在する場合に拒否し、無効化チェーンのヒントを表示。`claude plugin enable` は推移的依存を自動で有効化。
  ([code.claude.com](https://code.claude.com/docs/en/changelog))

- **`claude agents` に新フラグを追加**
  `--add-dir` / `--settings` / `--mcp-config` / `--plugin-dir` / `--permission-mode` / `--model` / `--effort` / `--dangerously-skip-permissions` でバックグラウンドセッションの細かい設定が可能に。
  ([releasebot.io](https://releasebot.io/updates/anthropic/claude-code))

- **MCP_TOOL_TIMEOUT のバグ修正**
  リモート HTTP / SSE MCP サーバー向けのタイムアウトが設定値に関わらず60秒に固定されていた問題を修正。
  ([code.claude.com](https://code.claude.com/docs/en/changelog))

- **ワークツリー非使用環境向けの新設定**
  `worktree.bgIsolation: "none"` を設定すると、バックグラウンドセッションがワークツリーを使わずにワーキングコピーを直接編集可能に。
  ([releasebot.io](https://releasebot.io/updates/anthropic/claude-code))
