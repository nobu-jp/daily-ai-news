## 💡 Claude Code / MCP 関連

- **`/rewind` コマンドサポート追加**
  Claude Codeで会話を巻き戻せる `/rewind` コマンドが追加。長いセッションでの試行錯誤が容易に。
  - 情報源: [Claude Code Updates June 2026 - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **リモートMCPツールのハング問題を修正**
  応答のないリモートMCPツール呼び出しが5分間ブロックしていた問題を修正。タイムアウト時にエラーで中断するように変更。環境変数 `CLAUDE_CODE_MCP_TOOL_IDLE_TIMEOUT` でタイムアウト時間の変更も可能に。
  - 情報源: [Claude Code Updates June 2026 - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **`--safe-mode` フラグ追加 / `/cd` コマンド追加**
  `--safe-mode`（CLAUDE_CODE_SAFE_MODE）でCLAUDE.md・プラグイン・スキル・フック・MCPサーバーを無効化してデバッグ起動が可能に。`/cd` でセッション中に作業ディレクトリを変更できるように。
  - 情報源: [Claude Code Changelog - claudefa.st](https://claudefa.st/blog/guide/changelog)

- **MCPエラーメッセージの改善**
  HTTP 404エラーでURLとMCP設定ファイルへの案内を表示するように改善。
  - 情報源: [Claude Code Updates June 2026 - Releasebot](https://releasebot.io/updates/anthropic/claude-code)
