## 💡 Claude Code / MCP 関連

- **MCP ツール検索（Lazy Loading）機能** — MCP サーバーのレイジーロードにより、コンテキスト使用量を最大 95% 削減。多数の MCP サーバーを同時利用してもコンテキスト上限を気にせず利用可能に。
  - 情報源: [Changelog - Claude Code Docs](https://code.claude.com/docs/en/changelog)

- **`/mcp` コマンド強化** — 接続サーバーのツール数を表示、ツールが 0 件のサーバーにフラグ表示。`MCP: workspace` が予約済みサーバー名となり、既存の同名サーバーはスキップ＋警告。再接続時の全ツール名フラッド通知を解消し、サーバープレフィックスで要約表示に変更。
  - 情報源: [Claude Code Updates - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **MCP サーバー自動リトライ** — 起動時に一時的なエラーが発生した MCP サーバーが、切断状態のままにならずに最大 3 回まで自動リトライするようになった。
  - 情報源: [Changelog - Claude Code Docs](https://code.claude.com/docs/en/changelog)

- **プラグイン読み込み拡張** — `--plugin-dir` が `.zip` アーカイブを受け付けるように。新フラグ `--plugin-url` でセッション中に URL からプラグインアーカイブを取得可能に。
  - 情報源: [Claude Code Updates - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **バグ修正** — `CLAUDE_CODE_SHELL_PREFIX` 設定時にスペース・シェルメタキャラクタを含む引数が壊れていた問題を修正。OAuth トークンの並列更新競合バグも解消。
  - 情報源: [Claude Code Changelog - claudefa.st](https://claudefa.st/blog/guide/changelog)
