- **Claude Opus 4.7 が正式リリース** — ソフトウェアエンジニアリングや複雑な長時間コーディングタスクの性能が向上。高解像度の画像認識も強化。価格は Opus 4.6 と同じ $5/$25 per MTok を維持。

  ([Releasebot - Claude Updates](https://releasebot.io/updates/anthropic/claude))

- **Claude Managed Agents に3つの新機能** — ①「Dreaming」：過去セッションを振り返りパターンを発見しエージェントが自己改善する記憶拡張機能。②マルチエージェントオーケストレーション：リードエージェントがタスクを分解し、専門エージェントに並列委任。③共有ファイルシステム上での協調動作。
  ([9to5Mac](https://9to5mac.com/2026/05/07/anthropic-updates-claude-managed-agents-with-three-new-features/))

- **Claude Code・Claude API の使用制限を大幅拡張** — Pro・Max・Team・Enterprise の全レートリミットを倍増。Pro/Max のピーク時間帯制限を撤廃。SpaceX の新コンピューティング設備稼働により Opus API 制限も引き上げ。
  ([Releasebot - Claude Code Updates](https://releasebot.io/updates/anthropic/claude-code))

## 💡 Claude Code / MCP 関連

- **`/mcp` コマンドが強化（5/4リリース）** — 接続サーバーのツール数を表示、ツール0件のサーバーを警告表示。起動時の一時エラーは最大3回まで自動リトライ。
  ([Claude Code Changelog](https://code.claude.com/docs/en/changelog))

- **「workspace」が予約済みサーバー名に** — 既存の同名 MCP サーバーはスキップされ警告が出るため、名前変更が必要。
  ([Releasebot - Claude Code Updates](https://releasebot.io/updates/anthropic/claude-code))

- **MCP 再接続時のノイズ削減** — 再接続のたびにツール名リストが全件流れていた問題を修正。サーバープレフィックスによるサマリー表示に変更。
  ([Claude Code Changelog](https://code.claude.com/docs/en/changelog))

- **重大バグ修正** — `CLAUDE_CODE_SHELL_PREFIX` 設定時にスペースやシェルメタ文字を含む引数が壊れていた問題を修正。10GB超のメモリリーク（stdio MCP サーバーが非プロトコルデータを stdout に書き込んだ際）も修正。
  ([Claude Code Changelog](https://code.claude.com/docs/en/changelog))

- **`claude project purge` コマンド追加（5/1リリース）** — プロジェクトの全 Claude Code 状態（トランスクリプト・タスク・ファイル履歴・設定エントリ）を削除。`--dry-run` / `-i` / `--all` オプション付き。
  ([Claude Code Changelog](https://code.claude.com/docs/en/changelog))
