## 💡 Claude Code / MCP 関連

- **Claude Code 広範な信頼性・ワークフロー更新**：
  - バックグラウンドセッション処理の改善（`/resume` コマンド対応、`--bg` セッションも一覧表示）
  - MCP・SDK の起動高速化（スロー MCP サーバーで最大2秒短縮）
  - ターミナルのグリッチ・クラッシュ・セッション復旧に関する多数の修正
  - 情報源: [Claude Code Updates (Releasebot)](https://releasebot.io/updates/anthropic/claude-code)

- **新しい `claude agents` フラグ追加**：バックグラウンドセッション向けに `--add-dir`・`--settings`・`--mcp-config`・`--plugin-dir`・`--permission-mode`・`--model`・`--effort`・`--dangerously-skip-permissions` など多数のフラグが追加され、細かな設定が可能に。
  - 情報源: [Claude Code Changelog (Releasebot)](https://releasebot.io/updates/anthropic/claude-code)

- **起動ハング問題を修正**：`api.anthropic.com` に到達できない環境（キャプティブポータル・ファイアウォール・VPN）で起動が最大75秒ハングしていた問題を修正。サイドチャネル API コールのタイムアウトを15秒に設定。
  - 情報源: [Claude Code Changelog (Claude Code Docs)](https://code.claude.com/docs/en/changelog)

- **Fast モードのデフォルトが Opus 4.7 に変更**：旧バージョン (Opus 4.6) に固定するには `CLAUDE_CODE_OPUS_4_6_FAST_MODE_OVERRIDE=1` を設定。
  - 情報源: [Claude Code Changelog (claudefa.st)](https://claudefa.st/blog/guide/changelog)
