## 💡 Claude Code / MCP 関連

- **MCPツール呼び出し 2分超で自動バックグラウンド化**: 長時間MCP処理がセッションをブロックしないよう改善。閾値は `CLAUDE_CODE_MCP_AUTO_BACKGROUND_MS` で設定可能。
  - 情報源: [Claude Code Updates - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **MCPエラーメッセージ改善**: サーバー設定に `url` のみで `type` が無い場合、`"type": "http"` を提案するより分かりやすいエラーに。
  - 情報源: [Claude Code Changelog - gradually.ai](https://www.gradually.ai/en/changelogs/claude-code/)

- **`/resume` セッションピッカー追加**: エージェントビューで `/resume` を入力すると、削除済みセッションも含む過去セッション一覧が表示され、バックグラウンドセッションとして再開可能。
  - 情報源: [Claude Code Updates - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **WebSearch・サブエージェント暴走防止キャップ追加**: セッションあたりWebSearch上限200回（`CLAUDE_CODE_MAX_WEB_SEARCHES_PER_SESSION`で変更可）、サブエージェントスポーン上限200回（`CLAUDE_CODE_MAX_SUBAGENTS_PER_SESSION`で変更可）。
  - 情報源: [Claude Code Updates - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **`/fork` バックグラウンドセッション対応・安全性強化**: WebSearch・サブエージェント・Bash実行のセーフガードを強化。MCP・ワークツリー・リモートセッション全般の安定性も向上。
  - 情報源: [Claude Code Changelog - code.claude.com](https://code.claude.com/docs/en/changelog)
