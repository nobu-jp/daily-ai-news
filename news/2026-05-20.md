# 🌅 AI朝刊 - 2026-05-20

## 🤖 Claude / Anthropic アップデート

- **Claude for Small Business 正式リリース** — QuickBooks・HubSpot・Google Workspace・Microsoft 365 など主要ビジネスツールに Claude が統合。給与計算・請求書発行・マーケティング・月次決算などのワークフローが利用可能に。
  [Anthropic News](https://www.anthropic.com/news)

- **Claude Managed Agents に3機能追加** — ① 「dreaming」（過去セッションを振り返り自己改善するメモリ機能、リサーチプレビュー）、② マルチエージェント・オーケストレーション（リードエージェントがタスクを分割し専門エージェントへ並列委譲）、③ 共有ファイルシステムを通じた協調コンテキスト共有。
  [9to5Mac](https://9to5mac.com/2026/05/07/anthropic-updates-claude-managed-agents-with-three-new-features/)

- **Developer Platform: Fast Mode & Cache Diagnostics** — Claude Opus 4.7 向けに Fast Mode が API でサポート開始（`fast-mode-2026-02-01` ベータヘッダー）。またキャッシュ診断（`diagnostics.previous_message_id`）がパブリックベータに移行し、プロンプトキャッシュのミス原因を詳細に取得可能に。
  [Claude API Docs](https://platform.claude.com/docs/en/release-notes/overview)

- **PwC との大型提携拡大** — PwC がグローバル全社員に Claude Code・Cowork を展開。米国で3万人の専門家が Claude 認定を取得。保険引受業務が従来10週間 → 10日に短縮。
  [Claude Updates - Releasebot](https://releasebot.io/updates/anthropic/claude)

- **Gates Foundation との $2億パートナーシップ** — Anthropic が4年間で2億ドルを拠出する Gates Foundation とのパートナーシップを発表。
  [Anthropic News](https://www.anthropic.com/news)

## 💡 Claude Code / MCP 関連

- **Tool Search がデフォルト有効化** — Claude Code において Tool Search 機能がデフォルト ON に。MCP サーバーを複数接続していても、セッション開始時にすべてのツールスキーマをロードせず必要時のみ取得するため、コンテキストウィンドウの肥大化を抑制。
  [Claude Code Updates - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **MCPサーバー登録数が9,400超に拡大** — 公式レジストリへの登録サーバー数が急増。Google Colab が独自 MCP サーバーを公開（2026年4月）するなど、Anthropic 以外のプラットフォームへの広がりが加速。
  [MCP's biggest growing pains for production use](https://thenewstack.io/model-context-protocol-roadmap-2026/)

- **MCPのプロダクション対応ロードマップ公開** — Linux Foundation 傘下（2025年12月〜）となった MCP プロジェクトが2026年版ロードマップを発表。エンタープライズ本番環境での信頼性向上を最優先課題に位置付け。
  [MCP Roadmap 2026 - The New Stack](https://thenewstack.io/model-context-protocol-roadmap-2026/)

- **VS Code への MCP 統合** — VS Code 側でも MCP (Model Context Protocol) の Claude Code 統合に関する Issue が進行中。
  [GitHub Issue #287660](https://github.com/microsoft/vscode/issues/287660)
