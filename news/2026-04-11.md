## 💡 Claude Code / MCP 関連

- **MCP 結果サイズ上限が500,000文字に拡張**: `_meta` 設定の `anthropic/maxResultSizeChars` を使用して最大50万文字まで対応。大規模ファイル・DBアウトプット・ログ処理のボトルネックが解消。
  - 情報源: [Claude Code April 2026 Update (Daily 1 Bite)](https://daily1bite.com/en/blog/ai-tutorial/claude-code-april-2026-update)

- **MCP 安定性の修正**:
  - セッション開始時にプラグインMCPサーバーが「接続中」のまま止まる問題を修正
  - サブエージェントが動的インジェクトされたサーバーのMCPツールを継承しない問題を修正
  - MCP OAuth の `authServerMetadataUrl` 設定が再起動後のトークンリフレッシュで無視される問題を修正（ADFS 等に影響）
  - 情報源: [Claude Code Changelog (GitHub)](https://github.com/anthropics/claude-code/blob/main/CHANGELOG.md)

- **MCP OAuth の強化**: RFC 9728 Protected Resource Metadata ディスカバリーに対応。Client ID Metadata Document（CIMD / SEP-991）のサポートも追加され、Dynamic Client Registration 非対応サーバーでも利用可能に。
  - 情報源: [Claude Code Docs - Changelog](https://code.claude.com/docs/en/changelog)

- **Claude Code の全般的アップデート**: チームオンボーディングガイド追加、リモートセッション設定の強化、Brief/Focus モードの改善、プラグイン・MCP ハンドリングの改善、Bash および MCP の安定性向上。
  - 情報源: [Claude Code Release Notes (Releasebot)](https://releasebot.io/updates/anthropic/claude-code)
