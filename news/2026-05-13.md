## 🤖 Claude / Anthropic アップデート

- **Claude Mythos Preview 公開 & Project Glasswing 始動**
  Anthropic が開発した最新モデル「Mythos Preview」がサイバーセキュリティ分野で圧倒的な性能を発揮。Firefox の未知脆弱性を数百回の試行で181回エクスプロイト成功（旧 Opus 4.6 は2回）。悪用リスクを抑えるため Apple・Amazon・JPMorgan Chase・Palo Alto Networks など米国企業に限定公開中。Anthropic は対策として **Project Glasswing** を立ち上げ、重要ソフトウェアの防御に Mythos を活用する取り組みに最大 $100M のクレジットと $400万 の直接寄付を投入。
  - [Claude Mythos Preview - red.anthropic.com](https://red.anthropic.com/2026/mythos-preview/)
  - [Anthropic Project Glasswing](https://www.anthropic.com/glasswing)
  - [Mythos サイバーセキュリティ懸念 - CNBC](https://www.cnbc.com/2026/05/08/anthropic-mythos-ai-cybersecurity-banks.html)

- **Claude Managed Agents に3つの新機能追加**
  - **Dreaming（ドリーミング）**: 過去セッションを振り返りパターンを学習し、エージェントが自己改善できる記憶拡張機能
  - **マルチエージェントオーケストレーション**: リードエージェントがタスクを分解し、各専門エージェント（個別モデル・プロンプト・ツール）へ並列委任、共有ファイルシステム上で協調動作
  - [Anthropic Updates - 9to5Mac](https://9to5mac.com/2026/05/07/anthropic-updates-claude-managed-agents-with-three-new-features/)

- **Claude Platform on AWS 提供開始**
  Anthropic 管理インフラ上で Claude API を AWS 経由（AWS 課金・IAM 認証）で利用可能に。Messages API・Files API・Batch API・Agent Skills・コード実行・ツール使用に対応。
  - [Anthropic Release Notes - Releasebot](https://releasebot.io/updates/anthropic)

- **Microsoft 365 アドイン GA & Outlook ベータ開始**
  Excel・PowerPoint・Word 向け Claude アドインが一般提供開始。Outlook は有料プラン向けパブリックベータに。アプリ間でコンテキストを保持し、開いているファイル間でリアルタイム編集同期が可能。

- **法律向け MCP コネクター 20+ 追加**
  リサーチ・契約・ディスカバリー・案件管理・法律支援など12の分野別プラグインを含む法律特化 MCP コネクターを20件以上リリース。

## 💡 Claude Code / MCP 関連

- **MCP 接続性の改善**
  - `/mcp` コマンドが接続サーバーのツール数を表示、0ツールで接続したサーバーを警告表示
  - MCP サーバーが起動時に一時エラーで失敗した場合、最大3回まで自動リトライ（従来は接続断のまま）
  - MCP stdio サーバーが環境変数 `CLAUDE_PROJECT_DIR` を受け取れるように（フックと統一）
  - `/mcp Reconnect` が再起動なしで `.mcp.json` の編集を反映、接続失敗時に HTTP ステータスと URL を表示
  - [Claude Code Docs - Changelog](https://code.claude.com/docs/en/changelog)

- **プラグイン機能の拡張**
  - `--plugin-dir` がディレクトリに加えて `.zip` アーカイブも受け入れ可能に
  - `--plugin-url` フラグで URL から直接プラグインアーカイブをフェッチ可能に

- **Hook システムの強化**
  - `PostToolUse` フックが全ツールで `hookSpecificOutput.updatedToolOutput` によるツール出力の置換に対応（従来は MCP のみ）

- **UI 改善**
  - ターミナルをオーバーフローするダイアログが矢印キー・PgUp/PgDn・マウスホイールでスクロール可能に
  - [Claude Code Changelog (claudefa.st)](https://claudefa.st/blog/guide/changelog)
  - [GitHub - anthropics/claude-code releases](https://github.com/anthropics/claude-code/releases)

- **AWS Kiro：MCP と連携したエージェント開発 IDE が進化**
  AWS の AI 開発 IDE「Kiro」が3機能を追加。①**Parallel Task Execution**（独立タスクの並列実行）、②**Quick Plan**（要件・設計・タスクを1パスで生成）、③**Requirements Analysis**（ニューロシンボリック AI で要件の矛盾・曖昧さをコード前に検出）。Claude Code と競合しつつも、仕様駆動開発という独自アプローチで差別化。
  - [SD Times - News Roundup May 13, 2026](https://sdtimes.com/ai/news-roundup-may-13-2026-aws-kiro-uipath-signadot/)
  - [SiliconANGLE - AWS Kiro](https://siliconangle.com/2026/05/12/aws-kiro-accelerates-software-development-proving-code-correctness-gets-work/)
