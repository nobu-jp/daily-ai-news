## 💡 Claude Code / MCP 関連

- **MCPツール呼び出しのバックグラウンド自動移行**：実行時間が2分を超えるMCPツール呼び出しは自動的にバックグラウンド処理へ移行。閾値は環境変数 `CLAUDE_CODE_MCP_AUTO_BACKGROUND_MS` で変更可能。
  - 情報源: [Claude Code Updates - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **`/fork` と `/resume` フローの強化**：`/fork` は会話を新しいバックグラウンドセッションへコピーし、並行作業が可能に。`/resume` はエージェントビューで過去セッション（削除済み含む）のピッカーを表示するよう改善。
  - 情報源: [What's new - Claude Code Docs](https://code.claude.com/docs/en/whats-new)

- **WebSearch・サブエージェント乱用防止のセーフガード追加**：セッションあたりのWebSearch上限（デフォルト200回）および サブエージェント起動上限（デフォルト200回）が追加され、無限ループ的な暴走を抑制。
  - 情報源: [Claude Code Updates - Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **安定性・ワークフロー改善の大型アップデート**：サブエージェントのテキストストリーミング、パーミッション・アップロード処理、バックグラウンドエージェントレポート、ターミナル描画の高速化など多数の修正。
  - 情報源: [Claude Code Changelog - gradually.ai](https://www.gradually.ai/en/changelogs/claude-code/)
