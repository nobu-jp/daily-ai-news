# 🌅 AI朝刊 - 2026-04-24

## 🤖 Claude / Anthropic アップデート

- **Claude Opus 4.7 リリース**: 高度なソフトウェアエンジニアリングタスクで Opus 4.6 から大幅改善。ビジョン機能も強化され、より高解像度な画像認識が可能に。料金は Opus 4.6 と同じ（入力 $5/Mトークン、出力 $25/Mトークン）。
  - 情報源: [Anthropic公式](https://www.anthropic.com/news/claude-opus-4-7)

- **品質低下問題を公式認定・修正**: Anthropic が3〜4月に発生した3つの品質低下の調査結果を公表。①3/4 に Claude Code の推論レベルを high→medium に引き下げ（4/7 に差し戻し）、②3/26 にキャッシュバグで思考セッションがリセットされ「物忘れ」状態に、③4/16 に簡潔化を意図したシステムプロンプト変更（4/20 に差し戻し）。
  - 情報源: [The Register](https://www.theregister.com/2026/04/23/anthropic_says_it_has_fixed/)

- **Claude Code の Pro プランからの一時除外テスト**: 新規 Pro ユーザーの約2%を対象に Claude Code を除外する小規模テストを実施中。既存の Pro・Max 契約者には影響なし。
  - 情報源: [The Register](https://www.theregister.com/2026/04/22/anthropic_removes_claude_code_pro/) / [wheresyoured.at](https://www.wheresyoured.at/news-anthropic-removes-pro-cc/)

- **Managed Agents 発表**: 長期エージェント作業向けのホスト型サービスを Claude Platform に導入。セッション・ハーネス・サンドボックスの安定したインターフェース、永続ステート、安全なツールアクセス、高速起動を提供。
  - 情報源: [Anthropic News](https://www.anthropic.com/news)

- **インライン可視化機能追加**: Claude がレスポンス内にカスタムチャート・ダイアグラム・ビジュアライゼーションを直接生成できるように。

- **広告なしコミットメント表明**: Anthropic は「広告インセンティブは真に役立つ AI アシスタントと相容れない」として Claude を広告なしで維持すると発表。
