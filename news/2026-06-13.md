## 💡 Claude Code / MCP 関連

**Claude Code v2.1.176（2026年6月12日リリース）**
- [公式Changelog](https://code.claude.com/docs/en/changelog)
- [Releasebot - Claude Code Updates](https://releasebot.io/updates/anthropic/claude-code)

**主な新機能・変更点：**

- **セッションタイトルの多言語対応** — タイトルが会話の言語で自動生成されるように。`language` 設定で言語を固定することも可能
- **Safe Mode 追加** — `--safe-mode` フラグでクリーンな診断ベースラインを取得可能。エンタープライズのポリシー設定は Safe Mode でも適用される
- **`/cd` コマンド追加** — セッションを新しい作業ディレクトリに移動できるコマンドを追加
- **ポストセッションフック追加** — セッション終了後に実行するフックをサポート
- **MCP ポリシー強化** — MCP の Policy 適用を厳格化。`/mcp` コマンドで未サインインの claude.ai コネクターを折りたたんで表示する改善も
- **モデル制限の保護強化** — `availableModels` の制限が `ANTHROPIC_DEFAULT_*_MODEL` 環境変数による迂回から保護されるように。`/fast` トグルがブロック済みモデルへの切り替えを拒否
- **Bedrock 認証の改善** — `awsCredentialExport` の認証情報が `Expiration` まで正しくキャッシュされるように
- **ファイルパーミッションルールの修正** — `Edit(src/**)` や `Read(~/.ssh/**)` などのワイルドカードパターンが正しく機能するように修正
- **並列ツール呼び出し改善** — Bash コマンドが失敗しても、同一バッチ内の他のツール呼び出しがキャンセルされなくなった
- **Auto Mode 改善** — Fable 5 で Opus 4.8 が利用不可の場合、最適な Opus モデルに自動フォールバック
