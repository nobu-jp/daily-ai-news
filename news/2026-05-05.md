## 💡 Claude Code / MCP 関連

- **`/mcp` コマンドの改善**: 接続済みサーバーのツール数を表示し、ツール0件で接続したサーバーをフラグ表示するようになった。
  情報源: [Changelog - Claude Code Docs](https://code.claude.com/docs/en/changelog)

- **MCP 再接続時の冗長表示を削減**: 再接続時にツール名一覧をフルで流すのではなく、サーバープレフィックス単位でサマリー表示するよう変更。

- **MCP サーバー名 `workspace` が予約済みに**: 同名サーバーはスキップされ警告が出る。命名衝突を防止。

- **MCP 起動時エラー時の自動リトライ**: 一時的なエラーで起動失敗した MCP サーバーが最大3回自動リトライするよう改善。

- **`--plugin-dir` が `.zip` アーカイブに対応**: ディレクトリに加えて ZIP 形式のプラグインアーカイブも受け付けるように。
  情報源: [Claude Code Updates by Anthropic - May 2026](https://releasebot.io/updates/anthropic/claude-code)
