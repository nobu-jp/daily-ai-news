## 💡 Claude Code / MCP 関連

- **`alwaysLoad` オプション追加** — MCP サーバー設定に `alwaysLoad: true` を指定すると、そのサーバーの全ツールがツール検索の遅延ロードをスキップして常時利用可能になる。
  - 出典: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

- **MCP 500K — ツール結果サイズ上限を大幅拡張** — `_meta["anthropic/maxResultSizeChars"]` アノテーションで上限を 500,000 文字に引き上げ可能に。大容量ファイル・DB 出力・ログのパススルーが実現。
  - 出典: [Claude Code April 2026 Update - Daily1Bite](https://daily1bite.com/en/blog/ai-tutorial/claude-code-april-2026-update)

- **PostToolUse フック強化** — 全ツールに対して `hookSpecificOutput.updatedToolOutput` でツール出力の差し替えが可能に（従来は MCP 専用）。さらにフックから MCP ツールを `type: "mcp_tool"` で直接呼び出せるようになった。
  - 出典: [Releasebot - Anthropic](https://releasebot.io/updates/anthropic)

- **`/mcp` コマンド改善** — 手動追加サーバーと同 URL の claude.ai コネクターが重複している場合、コネクターを表示しつつ重複削除のヒントを提示するように。
  - 出典: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)
