  - 情報源: [Releasebot - Claude Code June 2026](https://releasebot.io/updates/anthropic/claude-code)

- **Auto モードが Bedrock / Vertex / Foundry で利用可能に**: Opus 4.7 / 4.8 向けに Auto モードが主要クラウドプラットフォームへ展開。
  - 情報源: [Releasebot - Claude Code June 2026](https://releasebot.io/updates/anthropic/claude-code)

  - 情報源: [GitHub - claude-code releases](https://github.com/anthropics/claude-code/releases)

- **MCP サーバー改善**:
  - Stdio MCP サーバーのサブプロセスに `CLAUDE_CODE_SESSION_ID` と `CLAUDECODE=1` 環境変数を提供

  - `/mcp tools` リストで長い説明文・複数行のツール名が正しく描画されるよう修正
  - 情報源: [Claude Code Changelog](https://code.claude.com/docs/en/changelog) / [GitHub CHANGELOG](https://github.com/anthropics/claude-code/blob/main/CHANGELOG.md)
