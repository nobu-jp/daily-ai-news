# 🌅 AI朝刊 - 2026-05-12

## 🤖 Claude / Anthropic アップデート

- **Claude Managed Agents に3つの新機能が追加（5/6 発表）**
  - **Dreaming（ドリーミング）**：過去のエージェントセッションとメモリストアを定期的に分析し、パターン・ミス・共通の好みを抽出してエージェントが自己改善できる仕組み（Research Preview）。モデルの重みは変更せず、法律AI「Harvey」では導入後タスク完了率が約6倍に向上。
  - **Outcomes（アウトカムズ）**：エージェントの成果評価機能（パブリックベータ）。
  - **Multiagent Orchestration**：リードエージェントがタスクを分割し、各専門エージェント（独自モデル・プロンプト・ツール）に委任するマルチエージェント調整ツール（パブリックベータ）。
  - 情報源：[9to5Mac](https://9to5mac.com/2026/05/07/anthropic-updates-claude-managed-agents-with-three-new-features/) / [VentureBeat](https://venturebeat.com/technology/anthropic-introduces-dreaming-a-system-that-lets-ai-agents-learn-from-their-own-mistakes) / [claude.com ブログ](https://claude.com/blog/new-in-claude-managed-agents)

- **Claude Platform on AWS ローンチ**
  - AWS billing・IAM と統合したネイティブエンドポイントから Messages API・Files API・Message Batches API・Managed Agents・Agent Skills・コード実行・ツール使用がすべて利用可能に。
  - 情報源：[Anthropic リリースノート](https://platform.claude.com/docs/en/release-notes/overview)

- **利用可能モデル**：Claude Opus 4.7 / Sonnet 4.6 / Haiku 4.5（AWS 含む全プラットフォームで提供中）
  - 情報源：[Releasebot - Anthropic](https://releasebot.io/updates/anthropic)

## 📰 AI業界ニュース TOP3

1. **Stanford HAI「AI Index 2026」レポート公開 — SWE-benchが1年でほぼ100%到達**
   コーディングベンチマーク SWE-bench Verified のスコアが1年間で60%→約100%へ急伸。フロンティアモデルは博士レベルの科学問題・競技数学で人間を超えるパフォーマンスを達成。生成AIの企業導入率は88%、一般人口の採用率は3年で53%に達し、PCやインターネットより急速に普及。一方、Foundation Model Transparency Index は平均40点（前年58点）に低下し、AI関連インシデントは362件（前年233件）に増加。
   情報源：[Stanford HAI](https://hai.stanford.edu/news/inside-the-ai-index-12-takeaways-from-the-2026-report) / [IEEE Spectrum](https://spectrum.ieee.org/state-of-ai-index-2026)

2. **IBM調査：Chief AI Officer（CAIO）設置企業が76%に急増（前年26%から）**
   2,000社超を対象にした調査で、AIリーダーシップ職の設置が急加速。しかし AI採用の主な障壁として「技術的限界」ではなく「文化的課題」を挙げた回答者が93.2%に上り、組織変革の難しさが浮き彫りに。
   情報源：[CNBC](https://www.cnbc.com/2026/05/11/heres-how-artificial-intelligence-is-changing-boardrooms.html)

3. **Argentum AI、機関投資家向けAIインフラ融資事業を拡大**
   北米・欧州でのAIインフラプロジェクトを対象に、機関投資家・金融機関向けの構造化融資ソリューションを本格展開。エンタープライズAI採用の加速に伴うデータセンターなどの需要増に対応。
   情報源：[GlobeNewswire](https://www.globenewswire.com/news-release/2026/05/12/3293292/0/en/Andrew-Sobko-s-Argentum-AI-Expands-Focus-on-Institutional-AI-Infrastructure-Financing.html)

## 💡 Claude Code / MCP 関連

- **Claude Code 新機能（Code w/ Claude 2026 イベント、5/6 発表）**
  - **Agent View（Research Preview）**：全 Claude Code セッション（実行中・応答待ち・完了）を1つのリストで一覧管理できるビューが追加。
  - **`/goal` コマンド**：完了条件を設定するとClaudeが複数ターンにわたって目標達成まで継続動作するコマンドが追加。
  - その他：MCP・Remote Control・トランスクリプトナビゲーションの改善、ログイン・スクロール・プラグイン・Diff・エディタ統合の信頼性向上。
  - 情報源：[Claude Code Changelog](https://code.claude.com/docs/en/changelog) / [Simon Willison live blog](https://simonwillison.net/2026/May/6/code-w-claude-2026/)

- **MCP エコシステム拡大**
  - 公開 MCP サーバー数が **2,300件超** に到達。Claude・Cursor・Windsurf・VS Code など200以上のツールで採用。
  - MCP 改善：サーバー名 `workspace` が予約語に変更（既存サーバーは警告付きでスキップ）、再接続時のツール名一覧フラッド廃止、起動エラー時に最大3回自動リトライ、`/mcp` コマンドにツール数表示を追加。
  - **PostToolUse フック**が全ツールの出力を `hookSpecificOutput.updatedToolOutput` で置換可能に（従来はMCPのみ）。
  - Claude Code の ARR は **25億ドル**（早期 2026 時点）に到達。
  - 情報源：[Releasebot - Claude Code](https://releasebot.io/updates/anthropic/claude-code) / [Claude Code Changelog](https://code.claude.com/docs/en/changelog)
