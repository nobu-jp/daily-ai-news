## 🤖 Claude / Anthropic アップデート

- **Claude Opus 4.7 リリース**: 複雑な推論とエージェント型コーディング向けの最新最高スペックモデルが一般提供開始。価格は Opus 4.6 と同じ $5 / $25 per MTok。Fast モード（研究プレビュー）でも対応。

- **Claude for Small Business 発表**: QuickBooks・PayPal・HubSpot・Canva・Docusign・Google Workspace・Microsoft 365 などのツールと統合し、給与計算・請求書発行・営業・マーケティング・月次決算向けの自動化ワークフローを提供開始。（2026-05-13 発表）
  - [SiliconANGLE 報道](https://siliconangle.com/2026/05/13/anthropic-launches-claude-small-business-new-automation-workflows/)

- **Microsoft 365 統合 GA**: Claude for Excel・PowerPoint・Word が一般提供開始。Claude for Outlook は全有料プランで公開ベータへ移行。

- **Claude Platform on AWS**: Anthropic 管理インフラを通じて AWS 上で Claude API が利用可能に。AWS 請求・IAM 認証に対応し、Messages API・Files API・Message Batches API・Claude Managed Agents などすべての主要 API にアクセス可能。

- **Claude Managed Agents 3機能追加**（2026-05-07）: エージェント管理に関する3つの新機能が追加。
  - [9to5Mac 報道](https://9to5mac.com/2026/05/07/anthropic-updates-claude-managed-agents-with-three-new-features/)

## 💡 Claude Code / MCP 関連

- **MCP Tool Search（ツール遅延ロード）機能**: MCPツールをコンテキストに一括ロードせず、タスクに応じてClaudeが必要なツールを検索して呼び出す新しい仕組みが導入。コンテキスト効率が大幅改善。
  - [Claude Code MCP Docs](https://code.claude.com/docs/en/mcp)

- **MCP 出力トークン制御**: MCP ツールの出力が 10,000 トークンを超えた場合に警告を表示。`MAX_MCP_OUTPUT_TOKENS` 環境変数でデフォルト上限（25,000）を調整可能に。

- **Claude Code 大型アップデート**: フック・プラグインオプションの拡充、エージェント・セッション制御の改善、パーミッション強化、バックグラウンド処理と MCP ハンドリングの向上など多数の変更。新たに `ANTHROPIC_WORKSPACE_ID` 環境変数でワークロードアイデンティティフェデレーションに対応。
  - [Claude Code Changelog](https://claudefa.st/blog/guide/changelog)

- **MCP エコシステム 500+ サーバーへ拡大**: 2025年初頭の数十件から、2026年4月時点で公開 MCP サーバーが 500 件超に急増。Guidepoint が Claude 向け MCP を正式ローンチし、専門家インサイトをAI研究ワークフローに統合。
  - [MCP Bundles: State of MCP Clients (May 2026)](https://www.mcpbundles.com/blog/state-of-mcp-clients)
  - [Guidepoint MCP on Claude](https://article.wn.com/view-electricity/2026/05/05/Guidepoint_Launches_MCP_on_Claude_Embedding_Trusted_Expert_I_l/)
