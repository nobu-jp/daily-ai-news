# 🌅 AI朝刊 - 2026-05-30

## 🤖 Claude / Anthropic アップデート

- **Claude Opus 4.8 リリース**（5/30）: Anthropicが新モデル「Claude Opus 4.8」を正式リリース。コーディング・推論・実務作業タスクで Opus 4.7 を上回る性能を実現。主な改善点：
  - Agentic coding スコア: 64.3% → **69.2%**
  - マルチ分野推論（ツール使用）: 54.7% → **57.9%**
  - Knowledge work スコア: 1753 → **1890**
  - 価格は Opus 4.7 と変わらず
  - 出典: [gHacks Tech News](https://www.ghacks.net/2026/05/30/anthropic-releases-claude-opus-4-8-with-effort-controls-and-dynamic-workflows-for-claude-code/)

- **Dynamic Workflows（研究プレビュー）開始**: Claude Code 向けに、数十〜数百のエージェントをバックグラウンドで並列実行できる「Dynamic Workflows」機能を研究プレビューとして提供開始。大規模・複雑なタスクへの対応が可能に。
  - 出典: [9to5Mac](https://9to5mac.com/2026/05/28/anthropic-upgrades-claude-with-new-opus-4-8-model-heres-whats-new/)

- **Messages API 改善**: `messages` 配列内に `system` エントリを受け付けるようになり、プロンプトキャッシュを破壊せずにタスク途中でClaudeの指示を更新できるようになった。
  - 出典: [Releasebot - Anthropic](https://releasebot.io/updates/anthropic)

- **法律向け MCP コネクタ 20種以上追加**: 法律事務所・法務部門向けに、リサーチ・契約・ディスカバリー・案件管理・法律支援など横断する 20+ の法律系 MCP コネクタと 12 種のプラグインを公開。
  - 出典: [Releasebot - Claude](https://releasebot.io/updates/anthropic/claude)

- **Microsoft 365 アドイン GA**: Excel・PowerPoint・Word 向けの Claude アドインが一般提供（GA）に。Outlook は有料プランで公開ベータ開始。
  - 出典: [Releasebot - Anthropic](https://releasebot.io/updates/anthropic)
