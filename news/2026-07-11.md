## 💡 Claude Code / MCP 関連

- **Claude Code Desktop v2.1.206 — ブラウザパネル追加**: デスクトップアプリに組み込みブラウザが追加。Claude がドキュメントやデザインモックアップなどを直接操作できるように。ショートカットは macOS: `Cmd+Shift+B` / Windows: `Ctrl+Shift+B`。
  - 情報源: [jls42.org](https://jls42.org/en/news/ia-actualites-10-jul-2026) / [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

- **MCP バグ修正**: `--mcp-config` または `.mcp.json` で設定したMCPサーバーがサーバーごとの `request_timeout_ms` を無視し、デフォルト60秒でタイムアウトしていた問題を修正。長時間実行のMCPツールが正常に動作するように。
  - 情報源: [Releasebot - Claude Code](https://releasebot.io/updates/anthropic/claude-code)

- **「Claude Browser」MCP サーバー名を予約**: "Claude Preview" と並んで "Claude Browser" がシステム予約名に指定された。ユーザー設定のMCPサーバーでこれらの名前は使用不可。
  - 情報源: [Releasebot - Claude Code](https://releasebot.io/updates/anthropic/claude-code)

- **`claude mcp add-from-claude-desktop` の修正**: サーバー名に非対応文字が含まれる場合に処理がフリーズする問題を修正。無効な名前はスキップして報告し、残りのサーバーはインポートを継続するように改善。
  - 情報源: [Claude Code Changelog](https://claudefa.st/blog/guide/changelog)

- **Bedrock/Vertex AI/Foundry でのautoモードがデフォルトに**: Claude Codeがこれらのプラットフォームでも自動モードをデフォルト設定に。BedRockはClaude Opus 4.8に更新済み。
  - 情報源: [Releasebot - Claude Code](https://releasebot.io/updates/anthropic/claude-code)
