## 💡 Claude Code / MCP 関連

- **Claude Code セーフモード追加** — 破損した設定を分離するための`safe mode`が導入。設定ファイルに問題がある場合でも安全に起動可能。([Claude Code Changelog](https://code.claude.com/docs/en/changelog))
- **ネストされたサブエージェントに対応** — エージェントが自分自身のサブエージェントをスポーンできるようになり、より深いタスク分解が可能に。([Claude Code Changelog](https://code.claude.com/docs/en/changelog))
- **Managed AgentsがプライベートMCPサーバーへ接続可能に** — Managed Agentsが自分のコンテナ内でBashコマンド・ファイル操作・コード実行をサンドボックス実行できるようになり、プライベートMCPサーバーへの接続もサポート。([Releasebot Claude Code](https://releasebot.io/updates/anthropic/claude-code))
- **`Tool(param:value)` 権限ルール** — ツールの入力パラメータに基づいてマッチングする新しい権限ルール構文が追加。細かいアクセス制御が可能に。([Claude Code Changelog](https://code.claude.com/docs/en/changelog))
- **MCP強化** — HTTP/SSEサーバーの再接続機能向上、OAuthリフレッシュトークン保護強化、ページネーション対応改善。([Claude Code Changelog](https://code.claude.com/docs/en/changelog))
