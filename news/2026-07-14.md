## 🤖 Claude / Anthropic アップデート

- **Claude Cowork がモバイル・Web全面展開（7/7）**
  iOS・AndroidアプリおよびWeb版で「Claude Cowork」が利用可能に。セッション・ファイルのデバイス間同期、バックグラウンド作業、スケジュール設定タスク、モバイル承認機能などを搭載。

- **Claude Sonnet 5 が一般公開**
  ネイティブ 1M トークンコンテキストウィンドウを備えた Claude Sonnet 5 が正式提供開始。2026年8月31日まで入力100万トークンあたり2ドル・出力10ドルのプロモーション価格で利用可能。
  ([Anthropic Release Notes](https://releasebot.io/updates/anthropic))

- **Claude Fable 5 のグローバルアクセスが復活（7/1）**
  米国政府の輸出規制により一時停止していた Claude Fable 5 および Mythos 5 が規制解除され、7月1日からグローバルで再公開。
  ([Releasebot - Anthropic Updates](https://releasebot.io/updates/anthropic))

- **Microsoft 365 コネクタに書き込み機能を追加**
  メール下書き・送信、カレンダーイベント管理、OneDrive/SharePoint でのファイル作成・更新が Claude から直接実行可能に。
  ([Claude Platform Docs](https://docs.anthropic.com/ja/release-notes/overview))

- **Claude for Government ベータ版提供開始**
  政府機関向けの Claude for Government がベータ版として公開。Anthropic が契約・請求を直接受け持つため、別途クラウドプロバイダーとの契約が不要。
  ([Releasebot - Anthropic Updates](https://releasebot.io/updates/anthropic))

## 💡 Claude Code / MCP 関連

- **`claude mcp login` コマンドが追加**
  インタラクティブな `/mcp` メニューに依存せず、シェルから直接 MCP サーバーへ認証できる `claude mcp login` コマンドが実装。`claude mcp logout` による認証情報クリアもサポート。
  ([Claude Code Docs - What's New](https://code.claude.com/docs/ja/whats-new))

- **シェルモードが自動応答に対応**
  シェルモードで `npm test` などを実行すると、2回目のプロンプトなしにコマンド出力への説明・分析が得られるようになった。
  ([Claude Code Docs - What's New](https://code.claude.com/docs/ja/whats-new))

- **MCP セッション横断データ保持（最大 500,000 レコード）**
  MCP サーバーからの大規模な結果をセッションをまたいで保持できるように。大規模DBクエリ結果や API レスポンスの再利用が可能になり、長期タスクの効率が向上。
  ([Claude Code MCP 2026年版ガイド - crystal-method](https://crystal-method.com/blog/claude-code-mcp/))

- **MCP は Linux Foundation 傘下で業界標準へ**
  2025年12月に Linux Foundation へ寄贈されたMCPは、ベンダー中立の標準として普及が加速。kintone・Google Drive・Slack・Notion・GitHub などへの直接接続が標準的な開発スタイルとなりつつある。
  ([GitHub MCP ガイド 2026年7月 - genai-ai.co.jp](https://genai-ai.co.jp/ai-kanri/blog/cc-github-mcp-guide/))
