## 💡 Claude Code / MCP 関連

### Claude Code v2.1.202（7月6日リリース）
- **動的ワークフロースケール設定**: `/config` でワークフローのエージェント数規模（small/medium/large）を選択可能に
- **OpenTelemetry強化**: ワークフロー生成エージェントに `workflow.run_id` / `workflow.name` 属性を追加
- **`/review <pr>` を高速シングルパスに戻した**（マルチエージェントレビューは `/code-review <level> <pr#>` で利用可能）
- バグ修正多数: Ctrl+R 履歴検索のクラッシュ修正、Remote Control コマンド失敗修正、SSH サインインURL の折り返し問題修正、音声入力リトライループ修正など
- 情報源: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

### Claude Code v2.1.200（7月3日リリース）
- **デフォルト権限モードが「Manual」に変更**: CLI・--help・VS Code・JetBrains すべてで適用
- **AskUserQuestion がデフォルトで自動継続しなくなった**: アイドルタイムアウトは `/config` でオプトイン可能
- バックグラウンドセッションがスリープ/復帰後に停止していた問題を修正
- 情報源: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

### MCP関連修正
- リモートMCPツール呼び出しが5分間応答なしの場合、無限ブロックではなくエラーで中断するように変更
- MCPサーバー設定に `url` のみで `type` が未指定の場合のエラーメッセージを改善（`"type": "http"` を提案するガイダンスを表示）
- MCPサーバーが認証を必要とする場合に起動時通知と `/mcp` へのリンクを追加
- 情報源: [Releasebot Claude Code](https://releasebot.io/updates/anthropic/claude-code)
