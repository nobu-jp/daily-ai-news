## 📰 AI業界ニュース TOP3

### 1. Google、Gemini 3.5 Pro の3度目の遅延でAlphabet株4.4%下落・時価総額2000億ドル消失
Bloombergの報道によれば、Google の主力AIモデル「Gemini 3.5 Pro」がコーディング性能の向上を目指す開発の難航により、予定より数カ月遅れている。これはGeminiにとって3度目の遅延となり、「1度は規律の問題、3度は構造的問題」との見方が広がっている。AlphabetはNASDAQ市場で4.4%下落し、時価総額約2000億ドルが失われた。また同週には DeepMind の上級研究者4名が離職（Noam ShazeerはOpenAIへ、ノーベル賞受賞者John JumperはAnthropicへ）。
（[Bloomberg](https://www.bloomberg.com/news/articles/2026-07-16/google-gemini-launch-delayed-as-tech-falls-short-of-internal-goals) / [Yahoo Finance](https://finance.yahoo.com/technology/ai/articles/alphabet-stock-falls-report-gemini-184859374.html)）

### 2. 世界AI会議（上海）が閉幕 — 習近平が「世界AI協力機構」を発足
4日間の開催を経て2026年世界AI会議（WAIC）が7月20日に閉幕。習近平国家主席が初めて基調講演に立ち、パキスタン・ロシア・カザフスタンなど29カ国を創設メンバーとする「世界人工知能協力機構（WAICO）」が設立された。AI覇権争いにおける中国の影響力拡大を示す動きとして注目を集めている。
（[Al Jazeera](https://www.aljazeera.com/news/2026/7/17/chinas-xi-jinping-launches-new-ai-alliance-what-is-it)）

### 3. EUがGoogleにAndroidのAIアシスタント開放を命令 — 競合AIへの検索データ共有も
欧州委員会がデジタル市場法（DMA）に基づく拘束力のある要件を採択。Googleは Android端末を競合AIアシスタントに開放し、AI開発者を含む競合他社に検索データを提供することが義務付けられた。Androidの相互運用性は2027年7月まで、検索データ共有は2027年1月からの実施を予定。
（[UN News](https://news.un.org/en/story/2026/07/1167862) / [AI Weekly](https://aiweekly.co/ai-news-today)）

## 💡 Claude Code / MCP 関連

### Claude Code v2.1.215（2026年7月19日リリース）
（[公式チェンジログ](https://code.claude.com/docs/en/changelog) / [GitHub CHANGELOG](https://github.com/anthropics/claude-code/blob/main/CHANGELOG.md)）

**スキル実行の変更**
- `/verify` と `/code-review` はClaudeが自動実行しなくなった。明示的なコマンド入力が必要。

**セキュリティ・権限管理の強化**
- Windows PowerShell 5.1 セッションでの権限確認バイパスを修正
- 10,000文字を超えるコマンドは自動承認されず、手動確認が必要に
- `docker` コマンドでデーモンリダイレクトフラグを使用する場合に権限プロンプトを表示

**バックグラウンドエージェント**
- `/fork` コマンドで会話を新しいバックグラウンドセッションにコピー可能に
- セッション全体の WebSearch 呼び出しにデフォルト200回の上限を設定（`CLAUDE_CODE_MAX_WEB_SEARCHES_PER_SESSION` で調整可能）

**MCP 改善**
- MCP ツール呼び出しが2分以上かかる場合、自動的にバックグラウンドへ移行（`CLAUDE_CODE_MCP_AUTO_BACKGROUND_MS` で閾値を変更・無効化可能）
- 公開アーティファクトからMCPコネクタのライブデータを呼び出せる機能を追加（ダッシュボードがスナップショットではなくリアルタイムデータを表示）

**UI・アクセシビリティ**
- スクリーンリーダーモード追加（`claude --ax-screen-reader` またはセッション設定から有効化）
- `/resume` コマンドでセッション選択ピッカーを表示
- セッション内にライブの経過時間カウンターを追加
