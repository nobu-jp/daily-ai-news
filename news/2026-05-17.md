# 🌅 AI朝刊 - 2026-05-17

## 🤖 Claude / Anthropic アップデート

- **Claude for Small Business 正式ローンチ**
  QuickBooks・PayPal・HubSpot・Canva・Docusign・Google Workspace・Microsoft 365 など主要ビジネスツールと統合済みのワークフローを提供開始。
  ([Anthropic News](https://www.anthropic.com/news))

- **Claude for Microsoft 365 が一般提供（GA）へ**
  Excel・PowerPoint・Word 向けアドインが GA に移行。Outlook は有料プラン向けにパブリックベータ公開中。アプリをまたいだ会話コンテキストの保持と、開いているファイル間のリアルタイム同期が特徴。
  ([Anthropic News](https://www.anthropic.com/news))

- **Claude Managed Agents に3つの新機能を追加**
  マルチエージェントオーケストレーション強化。リードエージェントがタスクを分割し、専用モデル・プロンプト・ツールを持つスペシャリストエージェントへ並列委譲できる仕組みに。記憶機能「dreaming」も導入。
  ([9to5Mac](https://9to5mac.com/2026/05/07/anthropic-updates-claude-managed-agents-with-three-new-features/))

- **Claude Mythos Preview — Project Glasswing で限定公開**
  サイバーセキュリティ向け最強モデル「Claude Mythos」はパブリックリリースせず、約50の審査済み組織のみ利用可能な「Project Glasswing」として公開。1億ドルの利用クレジットを防衛的サイバーセキュリティ用途に提供。17年・27年物のゼロデイ脆弱性を自律的に発見・エクスプロイトできる能力が確認されており、広範公開を見送った経緯あり。
  ([red.anthropic.com](https://red.anthropic.com/2026/mythos-preview/) / [Dark Reading](https://www.darkreading.com/cybersecurity-operations/anthropic-mythos-cyber-what-comes-next))

- **Claude 利用制限の変更（厳格化）**
  Pro・Max プランの長時間エージェントループが以前より早くキャップに達するよう調整。OpenAI が重量ユーザー向けに安価なトークンを提供する中でのポリシー変更として注目。
  ([fazm.ai](https://fazm.ai/t/anthropic-claude-update-may-2026))

## 💡 Claude Code / MCP 関連

- **Claude Code の週次利用上限を50%増量（〜7/13まで）**
  Anthropic が競合 Codex への対抗策として、Claude Code の週次利用制限を7月13日まで50%引き上げ。
  ([pasqualepillitteri.it](https://pasqualepillitteri.it/en/news/2494/claude-code-weekly-limits-50-percent-anti-codex-anthropic-2026))

- **Fast モードが Opus 4.7 をデフォルトに変更**
  Claude Code の Fast モードが Opus 4.6 から Opus 4.7 へアップグレード。
  ([Releasebot](https://releasebot.io/updates/anthropic/claude-code))

- **MCP バグ修正: MCP_TOOL_TIMEOUT が正常動作するように**
  リモート HTTP / SSE MCP サーバーへのリクエストが設定値を無視して60秒で打ち切られていた問題を修正。
  ([Claude Code Changelog](https://code.claude.com/docs/en/changelog))

- **`workspace` がMCPの予約済みサーバー名に**
  同名の既存サーバーはスキップされ警告が出る。`/mcp` コマンドでは接続中サーバーのツール数と、ツール0件のサーバーがフラグ表示されるようになった。
  ([Claude Code Changelog](https://code.claude.com/docs/en/changelog))

- **プラグイン依存関係の強制管理を追加**
  `claude plugin disable` は依存している他のプラグインが有効な場合に拒否（コピペ可能な無効化チェーンのヒント付き）。`claude plugin enable` は推移的依存関係を自動で有効化。
  ([Releasebot](https://releasebot.io/updates/anthropic/claude-code))

- **エンタープライズ向け MCPツールスコーピング・監査ログ強化**
  企業環境での MCP ゲートウェイを通じたツールスコープ管理、開発者ごとのレート制限、監査ログ機能が拡充。
  ([TrueFoundry Blog](https://www.truefoundry.com/blog/claude-code-enterprise-mcp-gateway))

- **法律業界向け 20以上の新規 MCP コネクタをリリース**
  リサーチ・契約・ディスカバリー・案件管理・法律支援など12の専門領域プラグインと合わせて展開。
  ([Anthropic News](https://www.anthropic.com/news))
